# Plan: 41.3 Text/Link Color Inversion for Dark Backgrounds

## Context

When `bgcolor_propagator.py` (Phase 41.2) injects a dark bgcolor on a text section, the text and link colors remain dark — making content unreadable. Phase 41.3 pairs bgcolor propagation with automatic text/link color inversion.

Two integration points:
1. **During conversion** — `node_to_email_html()` already auto-inverts text color via `parent_bg` (converter.py:716-717). Links use `color:inherit` and inherit correctly. **No changes needed here.**
2. **Post-propagation** — `bgcolor_propagator.py` injects bgcolor AFTER conversion. Text/link colors inside those sections are NOT updated. **This is the gap.**

```python
_DARK_LUMINANCE_THRESHOLD = 0.4

# Dark text colors to replace with white when on dark backgrounds
_DARK_TEXT_COLORS_RE = re.compile(
    r'color\s*:\s*(#(?:0[0-9a-fA-F]|1[0-9a-fA-F]|2[0-9a-fA-F]|3[0-9a-fA-F]|4[0-4])[0-9a-fA-F]{4})\b',
)
```

Simpler approach — replace any `color:` that is itself dark (lum < 0.4) with white:

```python
_INLINE_COLOR_RE = re.compile(
    r"(color\s*:\s*)(#[0-9a-fA-F]{3,6})",
    re.IGNORECASE,
)


def _invert_text_colors(section_html: str, bgcolor: str) -> str:
    """Override dark text/link colors to white when bgcolor is dark.

    Scans inline ``color:`` styles in the section HTML. If the text color
    has low luminance (dark-on-dark), replaces it with ``#ffffff``.
    Leaves light text colors untouched (they already contrast).
    """
    bg_lum = _relative_luminance(bgcolor)
    if bg_lum >= _DARK_LUMINANCE_THRESHOLD:
        return section_html

    def _replace_dark_color(m: re.Match[str]) -> str:
        prefix = m.group(1)  # "color:"
        hex_val = m.group(2)
        text_lum = _relative_luminance(hex_val)
        if text_lum < _DARK_LUMINANCE_THRESHOLD:
            return f"{prefix}#ffffff"
        return m.group(0)

    return _INLINE_COLOR_RE.sub(_replace_dark_color, section_html)
```

### Step 3: Handle `color:inherit` on links

Links rendered by `_render_style_runs()` use `color:inherit` — they inherit from the parent `<td>`. Since `_invert_text_colors()` fixes the `color:` on the `<td>`, links with `color:inherit` will pick up the inverted color automatically. No regex needed for `color:inherit`.

However, links with explicit dark colors (e.g. `color:#0066cc`) on dark backgrounds need inversion. The regex already handles this because it matches all `color:#hex` patterns.

### Step 4: Handle VML button fallbacks

VML buttons use `fillcolor="{bg}"` and `color:{text}` in the `<center>` tag. The `_INLINE_COLOR_RE` regex matches `color:{text}` in VML too. The `fillcolor` attribute uses the button's own bg, not the section bg, so it needs no change.

Verify: VML `<center style="...color:{text_color};">` — if `text_color` is dark and the section bgcolor is dark, the regex replaces it with white. Correct behavior.

### Step 5: Add tests to `test_bgcolor_propagator.py`

```python
class TestTextColorInversion:
    """Tests for _invert_text_colors()."""

    def test_dark_bg_inverts_dark_text(self):
        html = '<td style="color:#000000;">Hello</td>'
        result = _invert_text_colors(html, "#0252B5")  # lum ≈ 0.10
        assert 'color:#ffffff' in result

    def test_dark_bg_preserves_light_text(self):
        html = '<td style="color:#ffffff;">Hello</td>'
        result = _invert_text_colors(html, "#0252B5")
        assert 'color:#ffffff' in result  # unchanged

    def test_light_bg_preserves_dark_text(self):
        html = '<td style="color:#000000;">Hello</td>'
        result = _invert_text_colors(html, "#ffffff")  # lum = 1.0
        assert 'color:#000000' in result  # unchanged

    def test_link_explicit_dark_color_inverted(self):
        html = '<a href="#" style="color:#0066cc;">Link</a>'
        result = _invert_text_colors(html, "#1a1a2e")  # very dark
        assert 'color:#ffffff' in result

    def test_link_inherit_unchanged(self):
        """Links with color:inherit are not touched (they inherit from td)."""
        html = '<a href="#" style="color:inherit;">Link</a>'
        result = _invert_text_colors(html, "#0252B5")
        assert 'color:inherit' in result

    def test_multiple_elements_all_inverted(self):
        html = (
            '<td style="color:#333333;">Text</td>'
            '<a href="#" style="color:#0066cc;">Link</a>'
        )
        result = _invert_text_colors(html, "#0252B5")
        assert result.count('color:#ffffff') == 2

    def test_vml_center_text_inverted(self):
        html = '<center style="font-size:16px;color:#000000;">CTA</center>'
        result = _invert_text_colors(html, "#0252B5")
        assert 'color:#ffffff' in result

    def test_threshold_boundary_0_4(self):
        """Luminance exactly at threshold: no inversion."""
        # #808080 has luminance ≈ 0.216 (dark) → inverts
        # #999999 has luminance ≈ 0.323 (dark) → inverts
        # #b3b3b3 has luminance ≈ 0.462 (light) → no inversion
        html = '<td style="color:#000000;">Text</td>'
        result_dark = _invert_text_colors(html, "#333333")   # lum ≈ 0.03
        result_light = _invert_text_colors(html, "#b3b3b3")  # lum ≈ 0.46
        assert 'color:#ffffff' in result_dark
        assert 'color:#000000' in result_light
```
