# Plan: Task 1.2 Backend — Project Access Control for Workspace

## Context

Task 1.2 adds a project workspace page at `/projects/[id]/workspace`. The workspace is primarily a frontend layout (3 resizable panes: editor, preview, AI chat), but requires two backend capabilities:

1. **Project-level access control** — Currently `GET /projects/{id}` requires only `get_current_user` (any authenticated user can read any project). The workspace needs membership-based access: users must be a project member (or admin) to access the workspace.
2. **Project member listing** — The workspace needs to know who else is on the project (for future collaboration features and to display the user's role within the project).

The existing `ProjectMember` model and `ProjectRepository.get_members()` / `add_member()` methods exist but are not exposed via any route or used for access control.

```python
async def get_member(self, project_id: int, user_id: int) -> ProjectMember | None:
    result = await self.db.execute(
        select(ProjectMember).where(
            ProjectMember.project_id == project_id,
            ProjectMember.user_id == user_id,
        )
    )
    return result.scalar_one_or_none()
```

### Step 5: Add service methods

**File:** `app/projects/service.py`

Add `verify_project_access` — checks membership, allows admin bypass:

```python
async def verify_project_access(self, project_id: int, user: User) -> ProjectResponse:
    """Verify user has access to project. Returns project if allowed.

    Admin users can access any project. Other users must be a project member.
    """
    logger.info("projects.access_check_started", project_id=project_id, user_id=user.id)
    project = await self.projects.get(project_id)
    if not project or project.deleted_at is not None:
        raise ProjectNotFoundError(f"Project {project_id} not found")

    if user.role != "admin":
        member = await self.projects.get_member(project_id, user.id)
        if not member:
            logger.warning(
                "projects.access_denied",
                project_id=project_id,
                user_id=user.id,
            )
            raise ProjectAccessDeniedError(
                f"User does not have access to project {project_id}"
            )

    return ProjectResponse.model_validate(project)
```

Add `list_project_members`:

```python
async def list_project_members(self, project_id: int) -> list[ProjectMemberResponse]:
    logger.info("projects.members_list_started", project_id=project_id)
    project = await self.projects.get(project_id)
    if not project or project.deleted_at is not None:
        raise ProjectNotFoundError(f"Project {project_id} not found")
    members = await self.projects.get_members(project_id)
    return [ProjectMemberResponse.model_validate(m) for m in members]
```

```python
    current_user: User = Depends(get_current_user),

    """Get a project by ID. Validates user has access."""
```
