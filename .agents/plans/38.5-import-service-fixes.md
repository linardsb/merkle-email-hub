## Implementation Steps

### Bug 36 (CRITICAL) — `_get_direct_text` drops `<b>` content
**File:** `html_import/dom_parser.py:354-362`

Current code only gets `element.text` + `child.tail`. Fix: recurse into child elements whose tags are in `_INLINE_TAGS`:
```python
_INLINE_TAGS = frozenset({"b", "strong", "em", "i", "u", "s", "sub", "sup", "span", "a"})

def _get_direct_text(element: HtmlElement) -> str:
    parts: list[str] = []
    if element.text:
        parts.append(element.text)
    for child in element:
        if child.tag in _INLINE_TAGS:
            parts.append(_get_direct_text(child))  # recurse into inline tags
        if child.tail:
            parts.append(child.tail)
    return " ".join(p for p in parts if p.strip())
```

### Bug 37 (HIGH) — `_filter_structure` drops rich properties
**File:** `service.py:1673-1702`

Replace manual `DesignNode(...)` construction with `dataclasses.replace()`:
```python
from dataclasses import replace

def _filter_node(node: DesignNode) -> DesignNode | None:
    if node.id in id_set:
        return node
    filtered_children = [c for c in (_filter_node(ch) for ch in node.children) if c is not None]
    if filtered_children:
        return replace(node, children=filtered_children)
    return None
```
This preserves all 20+ fields automatically. Add `from dataclasses import replace` at top of file if not present.
