# Plan: 44.11 Prompt Injection Detection for Agent Inputs

## Context

Agents process external content (Figma text, imported HTML, briefs, knowledge docs). A prompt injection in any input could override SKILL.md instructions. `sanitize_html_xss()` strips XSS but does not detect prompt injection. This adds a scanner that runs on all text inputs before they reach agents.

### Step 2: Core Module — `app/ai/security/prompt_guard.py`

```python
"""Prompt injection detection for AI agent inputs."""
from __future__ import annotations

import re
from dataclasses import dataclass, field
from app.core.logging import get_logger

logger = get_logger(__name__)

@dataclass(frozen=True)
class InjectionScanResult:
    clean: bool
    flags: list[str] = field(default_factory=list)
    sanitized: str | None = None
```

**Pattern categories** (compiled at module level):

| Category | Examples | Action |
|----------|----------|--------|
| Instruction override | "ignore previous instructions", "disregard above", "forget your instructions" | Flag `instruction_override` |
| System prompt leak | "output your system prompt", "reveal your instructions", "show me your rules" | Flag `system_prompt_leak` |
| Role-play | "you are now", "act as", "pretend to be", "from now on you are" | Flag `roleplay_attempt` |
| Delimiter attack | 3+ consecutive `---`, `###`, `===`; XML-like `<system>`, `</instructions>` | Flag `delimiter_attack` |
| Context switch | "NEW CONVERSATION", "SYSTEM:", "Human:", "Assistant:" | Flag `context_switch` |

**`scan_for_injection(text: str, *, mode: str = "warn") -> InjectionScanResult`:**

1. Return early if text is empty → `InjectionScanResult(clean=True)`
2. Normalize: lowercase for matching, preserve original for sanitized output
3. Run all compiled regex patterns against normalized text
4. Collect flags from all matches
5. If no flags → `InjectionScanResult(clean=True)`
6. If flags found:
   - `mode="warn"` → log warning, return `InjectionScanResult(clean=False, flags=flags, sanitized=None)`
   - `mode="strip"` → remove matched segments from original text, return with `sanitized=<stripped>`
   - `mode="block"` → raise `PromptInjectionError` (from `app/core/exceptions.py`)
7. Log: `logger.warning("security.prompt_injection_detected", flags=flags, mode=mode, text_length=len(text))`

**`scan_fields(fields: dict[str, str | None], *, mode: str = "warn") -> dict[str, InjectionScanResult]`:**

Convenience function to scan multiple named fields. Returns dict of field_name → result.

### Step 3: Exception — `app/core/exceptions.py`

Add after existing exceptions:

```python
class PromptInjectionError(DomainValidationError):
    """Prompt injection detected in user input."""
    def __init__(self, flags: list[str]) -> None:
        self.flags = flags
        super().__init__(f"Prompt injection detected: {', '.join(flags)}")
```

### Step 4: Wire into Blueprint Engine — `app/ai/blueprints/engine.py`

In `_build_node_context()` after building context but before returning, ~line 680-690:

```python
from app.ai.security.prompt_guard import scan_for_injection
from app.core.config import get_settings

settings = get_settings()
if settings.security.prompt_guard_enabled:
    mode = settings.security.prompt_guard_mode
    # Scan high-risk user-supplied fields
    for field_name, value in [
        ("brief", brief),
        ("html", context.html if context else None),
    ]:
        if value:
            result = scan_for_injection(value, mode=mode)
            if not result.clean:
                logger.warning("security.prompt_injection_detected",
                    field=field_name, flags=result.flags, agent=node.agent_name)
                if mode == "strip" and result.sanitized is not None:
                    # Replace field value with sanitized version
                    if field_name == "brief":
                        brief = result.sanitized
                    elif field_name == "html" and context:
                        context.html = result.sanitized
    # Also scan metadata fields that contain user-originated content
    for meta_key in ("qa_failure_details", "graph_context"):
        val = context.metadata.get(meta_key) if context else None
        if isinstance(val, str):
            result = scan_for_injection(val, mode=mode)
            if not result.clean and mode == "strip" and result.sanitized is not None:
                context.metadata[meta_key] = result.sanitized
```

## Preflight Warnings

- `app/ai/blueprints/engine.py` `_build_node_context()` is a large method (~220 lines). Insert scan logic in a focused block, don't refactor the method.
- `context.html` may be `None` — guard with `if value:` before scanning.
- `context.metadata` dict values may be `list` not `str` — type-check with `isinstance(val, str)`.
- Pattern matching must be case-insensitive but not strip HTML entities — scan raw text, not decoded.
- HTML comments (`<!-- ... -->`) are a common hiding spot — ensure patterns match inside comments.

## Security Checklist

- No new endpoints — this is internal middleware, no auth/rate-limit needed
- `PromptInjectionError` maps to 422 via `DomainValidationError` handler — no internal type leakage
- Patterns are defensive (compiled regex) — no ReDoS risk with bounded quantifiers
- `scan_for_injection` is pure function with no side effects except logging
- Feature-gated with `SECURITY__PROMPT_GUARD_ENABLED` — can disable in emergencies
