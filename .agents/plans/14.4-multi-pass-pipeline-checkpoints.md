# Plan: 14.4 Multi-Pass Pipeline Checkpoints

## Context

The Scaffolder's `ScaffolderPipeline` runs 3 LLM passes: layout (sequential) → content + design (parallel). Each pass is an expensive LLM call. If pass 3 fails, passes 1+2 are wasted. Task 14.4 adds per-pass checkpointing so the pipeline can resume from the last completed pass instead of restarting from scratch. This builds on the blueprint-level checkpoint system from 14.1/14.2 but operates at a finer granularity (intra-node).

## Architecture Decision

**Separate from blueprint checkpoints.** The existing `CheckpointStore` (14.1) captures full `BlueprintRun` state snapshots after each node. Pipeline checkpoints are different — they store partial `EmailBuildPlan` components within a single node execution. Rather than force-fitting pipeline state into `CheckpointData`, we define a lightweight `PipelineCheckpoint` dataclass and a `PipelineCheckpointCallback` protocol that the engine provides to the pipeline.

**Storage reuse.** Pipeline checkpoints are stored in the same `blueprint_checkpoints` table but with compound node names (e.g., `scaffolder:layout`, `scaffolder:content_design`) to distinguish them from node-level checkpoints. This avoids a new migration while keeping data co-located with the run.

```python
"""Per-pass checkpointing for the Scaffolder 3-pass pipeline."""

from __future__ import annotations

from dataclasses import asdict, dataclass, field
from datetime import UTC, datetime
from typing import Any, Literal, Protocol, runtime_checkable

from app.core.logging import get_logger

logger = get_logger(__name__)

PassName = Literal["layout", "content_design"]


@dataclass(frozen=True)
class PipelineCheckpoint:
    """Snapshot of completed pass results within a pipeline execution."""

    run_id: str
    pass_name: PassName
    pass_index: int  # 0=layout, 1=content_design
    data: dict[str, Any]  # serialized pass output
    created_at: datetime = field(default_factory=lambda: datetime.now(UTC))


@runtime_checkable
class PipelineCheckpointCallback(Protocol):
    """Callback protocol for persisting pipeline pass checkpoints."""

    async def save_pass(self, checkpoint: PipelineCheckpoint) -> None:
        """Persist a pass checkpoint. Must be fire-and-forget safe."""
        ...

    async def load_passes(self, run_id: str) -> list[PipelineCheckpoint]:
        """Load all pass checkpoints for a pipeline run, ordered by pass_index."""
        ...


def serialize_layout_pass(
    template_name: str,
    reasoning: str,
    section_order: tuple[str, ...],
    fallback_template: str | None,
    section_decisions: tuple[Any, ...],
    slot_details: list[dict[str, object]],
) -> dict[str, Any]:
    """Serialize layout pass output for checkpoint storage."""
    return {
        "template_name": template_name,
        "reasoning": reasoning,
        "section_order": list(section_order),
        "fallback_template": fallback_template,
        "section_decisions": [asdict(sd) for sd in section_decisions],
        "slot_details": slot_details,
    }


def serialize_content_design_pass(
    slot_fills: tuple[Any, ...],
    design_tokens: Any,
) -> dict[str, Any]:
    """Serialize content + design pass outputs for checkpoint storage."""
    return {
        "slot_fills": [asdict(sf) for sf in slot_fills],
        "design_tokens": asdict(design_tokens),
    }
```

```python
async def _save_pass_checkpoint(
    self,
    pass_name: PassName,
    pass_index: int,
    data: dict[str, Any],
) -> None:
    """Fire-and-forget checkpoint save after a completed pass."""
    if self._checkpoint_cb is None:
        return
    try:
        from app.ai.agents.scaffolder.pipeline_checkpoint import PipelineCheckpoint

        checkpoint = PipelineCheckpoint(
            run_id=self._run_id,
            pass_name=pass_name,
            pass_index=pass_index,
            data=data,
        )
        await self._checkpoint_cb.save_pass(checkpoint)
        logger.info(
            "scaffolder.pass_checkpoint_saved",
            pass_name=pass_name,
            run_id=self._run_id,
        )
    except Exception:
        logger.warning(
            "scaffolder.pass_checkpoint_failed",
            pass_name=pass_name,
            run_id=self._run_id,
            exc_info=True,
        )
```

**2c.** Add `_load_pass_checkpoints` helper for resume:

```python
async def _load_pass_checkpoints(self) -> dict[str, dict[str, Any]]:
    """Load existing pass checkpoints for resume. Returns {pass_name: data}."""
    if self._checkpoint_cb is None or not self._run_id:
        return {}
    try:
        checkpoints = await self._checkpoint_cb.load_passes(self._run_id)
        return {cp.pass_name: cp.data for cp in checkpoints}
    except Exception:
        logger.warning(
            "scaffolder.pass_checkpoint_load_failed",
            run_id=self._run_id,
            exc_info=True,
        )
        return {}
```

**2d.** Modify `execute()` to checkpoint after each pass and support resume:

```python
async def execute(
    self,
    brief: str,
    brand_config: dict[str, object] | None = None,
    resume: bool = False,  # NEW
) -> EmailBuildPlan:
    brief = sanitize_prompt(brief)

    # Load existing checkpoints if resuming
    cached_passes: dict[str, dict[str, Any]] = {}
    if resume and self._checkpoint_cb is not None:
        cached_passes = await self._load_pass_checkpoints()

    # Pass 1: Layout selection
    if "layout" in cached_passes:
        layout_data = cached_passes["layout"]
        template_selection = TemplateSelection(
            template_name=layout_data["template_name"],
            reasoning=layout_data["reasoning"],
            section_order=tuple(layout_data.get("section_order", ())),
            fallback_template=layout_data.get("fallback_template"),
        )
        section_decisions = tuple(
            SectionDecision(**sd) for sd in layout_data.get("section_decisions", [])
        )
        slot_details = layout_data.get("slot_details", [])
        logger.info("scaffolder.layout_pass_resumed", template=template_selection.template_name)
    else:
        template_selection, section_decisions = await self._layout_pass(brief)
        template = self._resolve_template_for_slots(template_selection)
        slot_details = (
            [
                {"slot_id": s.slot_id, "slot_type": s.slot_type,
                 "max_chars": s.max_chars, "required": s.required}
                for s in template.slots
            ] if template else []
        )
        # Checkpoint after layout pass
        await self._save_pass_checkpoint(
            "layout", 0,
            serialize_layout_pass(
                template_selection.template_name,
                template_selection.reasoning,
                template_selection.section_order,
                template_selection.fallback_template,
                section_decisions,
                slot_details,
            ),
        )

    # Pass 2+3: Content + Design (parallel)
    if "content_design" in cached_passes:
        cd_data = cached_passes["content_design"]
        slot_fills = tuple(SlotFill(**sf) for sf in cd_data["slot_fills"])
        design_tokens = DesignTokens(**cd_data["design_tokens"])
        logger.info("scaffolder.content_design_pass_resumed")
    else:
        # Need template object for slot resolution (may already be resolved above)
        if "layout" in cached_passes:
            template = self._resolve_template_for_slots(template_selection)
            # Re-derive slot_details if not cached (backward compat)
            if not slot_details and template:
                slot_details = [
                    {"slot_id": s.slot_id, "slot_type": s.slot_type,
                     "max_chars": s.max_chars, "required": s.required}
                    for s in template.slots
                ]

        slot_fills, design_tokens = await asyncio.gather(
            self._content_pass(brief, template_selection, slot_details),
            self._design_pass(brief, brand_config),
        )
        # Checkpoint after content+design pass
        await self._save_pass_checkpoint(
            "content_design", 1,
            serialize_content_design_pass(slot_fills, design_tokens),
        )

    # ... rest of execute() unchanged (locked fills merge, plan assembly) ...
```

Note: The `resume` parameter defaults to `False` — existing callers are unaffected.

Add a module-level adapter function that bridges `CheckpointStore` → `PipelineCheckpointCallback`:

```python
def _make_pipeline_checkpoint_adapter(
    store: "CheckpointStore",
    blueprint_name: str,
) -> object:
    """Adapt blueprint CheckpointStore to PipelineCheckpointCallback interface."""
    from app.ai.agents.scaffolder.pipeline_checkpoint import PipelineCheckpoint
    from app.ai.blueprints.checkpoint import CheckpointData

    class _Adapter:
        async def save_pass(self, checkpoint: PipelineCheckpoint) -> None:
            data = CheckpointData(
                run_id=checkpoint.run_id,
                blueprint_name=blueprint_name,
                node_name=f"scaffolder:{checkpoint.pass_name}",
                node_index=checkpoint.pass_index + 100,  # offset to avoid collision with node-level
                status="pass_complete",
                html="",
                progress=[],
                iteration_counts={},
                qa_failures=[],
                qa_failure_details=[],
                qa_passed=None,
                model_usage={},
                skipped_nodes=[],
                routing_decisions=[],
                handoff_history=[],
            )
            # Store pipeline data in the state_json via override
            data_dict = {"pipeline_pass": checkpoint.pass_name, "pipeline_data": checkpoint.data}
            # We need to save via the store but with custom state_json
            # Use the store's underlying mechanism
            from app.ai.blueprints.checkpoint_models import BlueprintCheckpoint
            import hashlib

            row = BlueprintCheckpoint(
                run_id=checkpoint.run_id,
                blueprint_name=blueprint_name,
                node_name=f"scaffolder:{checkpoint.pass_name}",
                node_index=checkpoint.pass_index + 100,
                state_json=data_dict,
                html_hash=hashlib.sha256(b"").hexdigest(),
            )
            store._db.add(row)
            await store._db.commit()

        async def load_passes(self, run_id: str) -> list[PipelineCheckpoint]:
            from sqlalchemy import select
            from app.ai.blueprints.checkpoint_models import BlueprintCheckpoint

            stmt = (
                select(BlueprintCheckpoint)
                .where(
                    BlueprintCheckpoint.run_id == run_id,
                    BlueprintCheckpoint.node_name.like("scaffolder:%"),
                    BlueprintCheckpoint.node_index >= 100,
                )
                .order_by(BlueprintCheckpoint.node_index.asc())
            )
            result = await store._db.execute(stmt)
            rows = result.scalars().all()
            return [
                PipelineCheckpoint(
                    run_id=run_id,
                    pass_name=row.state_json["pipeline_pass"],
                    pass_index=row.state_json.get("pass_index", row.node_index - 100),
                    data=row.state_json["pipeline_data"],
                )
                for row in rows
            ]

    return _Adapter()
```

**Important:** This adapter accesses `store._db` directly. This is acceptable because:
- Both checkpoint levels share the same DB session lifetime
- The adapter is internal to the blueprint engine (never exposed to external callers)
- The alternative (adding methods to `CheckpointStore` protocol) would break the clean separation between node-level and pass-level concerns

## Data Flow

```
BlueprintEngine.run()
  └─ ScaffolderNode.execute()
       └─ _execute_structured()
            └─ ScaffolderPipeline(checkpoint_callback=adapter, run_id=run.run_id)
                 ├─ Pass 1: _layout_pass() → checkpoint("layout")
                 ├─ Pass 2+3: _content_pass() + _design_pass() → checkpoint("content_design")
                 └─ assembly (no checkpoint needed — deterministic)
```

On retry (iteration > 0):
```
ScaffolderPipeline.execute(resume=True)
  ├─ Load existing checkpoints → {"layout": {...}}
  ├─ Skip _layout_pass() — use cached TemplateSelection
  └─ Run only _content_pass() + _design_pass()
```
