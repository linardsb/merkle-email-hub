## Implementation

### `TokenRewriterService` (`app/connectors/token_rewriter.py`)

```python
@dataclass(frozen=True, slots=True)
class TokenRewriteResult:
    html: str
    source_esp: str
    target_esp: str
    tokens_rewritten: int
    warnings: tuple[str, ...]

class TokenRewriterService:
    async def rewrite(
        self,
        html: str,
        target_esp: ESPPlatform,
        source_esp: ESPPlatform | None = None,  # auto-detect if None
    ) -> TokenRewriteResult:
        ...
```

**Flow:**
1. If `source_esp` is None → `detect_and_parse(html)` from token_ir
2. Else → `parse_tokens(html, source_esp)`
3. If source == target → return passthrough (0 rewritten)
4. `emit_tokens(ir, html, target_esp)` → new HTML + warnings
5. Count tokens rewritten = len(ir.variables) + len(ir.conditionals) + len(ir.loops)
6. Return `TokenRewriteResult`

### Schemas (`sync_schemas.py` additions)

```python
class TokenRewriteRequest(BaseModel):
    html: str = Field(..., max_length=5_000_000)
    target_esp: str
    source_esp: str | None = None

class TokenRewriteResponse(BaseModel):
    html: str
    source_esp: str
    target_esp: str
    tokens_rewritten: int
    warnings: list[str]
```

### Route (`sync_routes.py` addition)

```
POST /api/v1/connectors/sync/rewrite-tokens
- Auth: require_role("developer")
- Rate limit: 30/minute
- Body: TokenRewriteRequest
- Response: TokenRewriteResponse
```

### Wire into `push_template()`

In `sync_service.py:push_template()`, after fetching HTML and before calling `provider.create_template()`:
- Detect source ESP from HTML tokens
- If source ≠ target connection's esp_type → auto-rewrite
- Log rewrite event: `connectors.token_rewrite.auto`
