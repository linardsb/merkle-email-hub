## Implementation Steps

### Step 1: Create `use-smart-polling.ts`

```typescript
// cms/apps/web/src/hooks/use-smart-polling.ts
import { useEffect, useState } from "react";

type VisibilityState = "visible" | "hidden" | "blurred";

export function useSmartPolling(baseInterval: number): number {
  const [visibility, setVisibility] = useState<VisibilityState>("visible");

  useEffect(() => {
    const onVisibilityChange = () => {
      setVisibility(document.hidden ? "hidden" : "visible");
    };
    const onFocus = () => setVisibility("visible");
    const onBlur = () => {
      if (!document.hidden) setVisibility("blurred");
    };

    document.addEventListener("visibilitychange", onVisibilityChange);
    window.addEventListener("focus", onFocus);
    window.addEventListener("blur", onBlur);

    return () => {
      document.removeEventListener("visibilitychange", onVisibilityChange);
      window.removeEventListener("focus", onFocus);
      window.removeEventListener("blur", onBlur);
    };
  }, []);

  if (baseInterval === 0) return 0;

  switch (visibility) {
    case "visible": return baseInterval;
    case "blurred": return Math.round(baseInterval * 1.5);
    case "hidden": return 0;
  }
}
```

**Design decisions:**
- Returns `number` (not function) — composes with SWR's function form: `refreshInterval: (data) => data?.done ? 0 : smartInterval`
- `baseInterval === 0` → always 0 (preserves "no polling" pattern)
- Three states: visible (full), blurred (1.5x slowdown), hidden (paused)
- SSR-safe: `document` access inside `useEffect` only
- No `useCallback` import needed (removed from user spec — unused)

### Step 2: Create `use-smart-polling.test.ts`

6 tests following existing patterns from `use-complex-hooks.test.ts`:

1. **visible → returns baseInterval** — default state, `renderHook(() => useSmartPolling(3000))` → `result.current === 3000`
2. **hidden → returns 0** — fire `visibilitychange` with `document.hidden = true` → `result.current === 0`
3. **blurred → returns 1.5x** — fire `blur` without hidden → `result.current === 4500`
4. **baseInterval 0 → always 0** — all visibility states return 0
5. **cleanup on unmount** — spy on `removeEventListener`, unmount, verify cleanup called
6. **SSR safety** — no `document` access during initial render (verify no errors when `typeof document === "undefined"`)

**Mocking approach:**
- `Object.defineProperty(document, "hidden", { value: true, writable: true })` for visibility state
- `fireEvent` or manual `document.dispatchEvent(new Event("visibilitychange"))` for events
- `window.dispatchEvent(new Event("focus"))` / `window.dispatchEvent(new Event("blur"))` for focus
