# Plan: Phase 24.7 — Frontend Builder Integration & Workspace

## Context
The visual builder (24.4), property panels (24.5), and bidirectional sync (24.6) exist as self-contained components but need deeper workspace integration. The builder view currently lives inside `EditorPanel` with basic Code/Builder/Split tabs — it needs builder-specific toolbar actions (device preview, client preview, QA, AI suggest, export), a view switcher with keyboard shortcut, an onboarding flow for non-technical users, QA section-linking, and Arrow Up/Down reorder shortcuts.

**Key constraint:** i18n was removed from the codebase. All user-visible text uses raw string literals — no `useTranslations()`.
