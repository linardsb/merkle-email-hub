# Plan: 43.1 Judge Correction Generator from Calibration Data

## Context

Phase 37.5 produced 540 human-labeled rows and calibration data with TPR/TNR metrics per criterion. FP/FN disagreements between judges and human labels are the judge's mistakes. This module extracts those mistakes, formats them as structured correction YAML, and outputs one file per agent at `traces/corrections/{agent}_judge_corrections.yaml`. Phase 43.2 will inject these into judge prompts.

### Step 1: Create `app/ai/agents/evals/judge_corrections.py`

```python
"""Generate structured correction YAML from calibration disagreements."""
from __future__ import annotations

import argparse
import json
from collections import defaultdict
from datetime import datetime, timezone
from pathlib import Path
from typing import Any

import yaml

from app.ai.agents.evals.calibration import load_human_labels, load_judge_verdicts
from app.ai.agents.evals.schemas import HumanLabel
from app.core.logging import get_logger

logger = get_logger(__name__)

# 7 LLM-judged agents (accessibility + outlook_fixer are deterministic)
LLM_JUDGED_AGENTS: frozenset[str] = frozenset({
    "scaffolder", "dark_mode", "content", "personalisation",
    "code_reviewer", "knowledge", "innovation",
})

MAX_CORRECTIONS_PER_CRITERION = 3
```

**Core functions:**

1. `_load_verdicts_with_reasoning(path: Path) -> dict[tuple[str, str], tuple[bool, str]]`
   - Read `{agent}_verdicts.jsonl`, return `{(trace_id, criterion): (passed, reasoning)}` lookup
   - Skip lines with `error != null`

2. `_extract_disagreements(labels: list[HumanLabel], verdict_lookup: dict[...]) -> list[dict[str, Any]]`
   - Match `(trace_id, criterion)` across labels and verdicts
   - Filter to FP (`judge_pass=True, human_pass=False`) and FN (`judge_pass=False, human_pass=True`)
   - Return list of `{trace_id, criterion, judge_passed, human_passed, judge_reasoning, error_type}` dicts
   - Sort by reasoning length ascending (shorter = less confident = more useful correction)

3. `_build_corrections(agent: str, disagreements: list[dict[str, Any]]) -> dict[str, Any]`
   - Group by criterion, take top `MAX_CORRECTIONS_PER_CRITERION` per criterion
   - Build YAML-serializable dict:
     ```yaml
     agent: scaffolder
     generated: "2026-04-01T..."
     correction_count: N
     corrections:
       - criterion: brief_fidelity
         type: false_positive  # or false_negative
         trace_id: scaff-003
         judge_said: PASS
         correct_answer: FAIL
         judge_reasoning: "..."
         pattern: "..."  # auto-generated: "Judge incorrectly {PASSed/FAILed} — {first 120 chars of reasoning}"
     ```
   - `pattern` field: `"Judge incorrectly {said PASS|said FAIL}. Reasoning was: {first 120 chars}"`

4. `generate_corrections(traces_dir: Path, output_dir: Path) -> list[Path]`
   - Main orchestrator. For each agent in `LLM_JUDGED_AGENTS`:
     - Load `{agent}_human_labels.jsonl` (skip if missing)
     - Load `{agent}_verdicts.jsonl` via `_load_verdicts_with_reasoning()`
     - Filter labels to `human_pass is not None`
     - Extract disagreements
     - Build corrections dict
     - Skip if 0 corrections
     - Write YAML to `output_dir/{agent}_judge_corrections.yaml`
   - Return list of written file paths
   - Log summary: `judge_corrections.generate_completed agent={agent} corrections={n}`

5. `def main() -> None` — CLI entry point
   - `argparse`: `--traces-dir` (default `traces/`), `--output` (default `traces/corrections/`)
   - Call `generate_corrections()`, print summary

**Factory helpers:**

```python
def _label(trace_id: str, agent: str, criterion: str, human_pass: bool | None) -> HumanLabel:
    return HumanLabel(trace_id=trace_id, agent=agent, criterion=criterion, human_pass=human_pass, notes="")

def _verdict_line(trace_id: str, agent: str, criteria: dict[str, tuple[bool, str]]) -> str:
    return json.dumps({
        "trace_id": trace_id, "agent": agent,
        "overall_pass": all(p for p, _ in criteria.values()),
        "criteria_results": [
            {"criterion": c, "passed": p, "reasoning": r} for c, (p, r) in criteria.items()
        ],
        "error": None,
    })
```
