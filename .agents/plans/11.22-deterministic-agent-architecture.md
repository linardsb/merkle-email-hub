# Plan: 11.22 — Deterministic Agent Architecture (Template-First Hybrid)

## Problem Statement

Hub agents currently have a **16.7% eval pass rate** (baseline 2026-03-09). The root cause: agents generate raw HTML via unconstrained LLM output, where every token is a chance for non-deterministic structural failures. MSO conditionals pass at 0%, accessibility at 8%, HTML preservation at 10%.

The workshop pattern (PIV loop) and Stripe's Minions architecture both demonstrate the same solution: **LLMs make decisions in structured JSON; deterministic code assembles the output**. This plan applies that pattern across all 10 Hub agents.

## Architecture Change

```
CURRENT (per agent):
  Brief → LLM generates raw HTML → post-process → QA gate

TARGET (per agent):
  Brief → LLM returns structured decisions (JSON) → deterministic assembler → cascading auto-repair → QA gate
```

The LLM becomes an **architect** (deciding what to do) rather than a **builder** (writing HTML). Deterministic code handles assembly, repair, and validation.

## Success Criteria

- Scaffolder structural pass rate: 0% → 95%+
- Overall eval pass rate: 16.7% → 85%+ (milestone 1), then 95%+ (milestone 2)
- Zero new agent test regressions
- All existing 1565+ tests continue passing
- Type checking clean (mypy + pyright 0 errors)

```python
class TemplateAssembler:
    """Deterministic HTML assembly from structured plan + golden template."""

    def __init__(self, registry: TemplateRegistry) -> None:
        self._registry = registry

    def assemble(self, plan: EmailBuildPlan) -> str:
        """
        Assemble final HTML from plan. 100% deterministic.

        1. Load golden template by plan.template.template_name
        2. Fill slots with plan.slot_fills (string replacement)
        3. Apply design tokens (CSS variable injection)
        4. Apply section decisions (show/hide, background overrides)
        5. Set preheader text
        6. Return complete HTML
        """

    def _apply_design_tokens(self, html: str, tokens: DesignTokens) -> str:
        """Replace CSS custom properties or inline style values."""

    def _apply_sections(self, html: str, sections: tuple[SectionDecision, ...]) -> str:
        """Show/hide sections, apply overrides."""
```

#### Step 2: Build ScaffolderPipeline

```python
# app/ai/agents/scaffolder/pipeline.py
class ScaffolderPipeline:
    """
    3-pass generation pipeline:
      Pass 1 (Layout):  Brief → TemplateSelection + SectionDecisions
      Pass 2 (Content): Brief + Template → SlotFills
      Pass 3 (Design):  Brief + Brand → DesignTokens

    Each pass uses structured output. Passes 2+3 can run in parallel.
    Per-slot retry on content failures.
    """

    async def execute(self, brief: str, brand_config: dict | None = None) -> EmailBuildPlan:
        # Pass 1: Layout selection (lightweight model — fast)
        template_selection = await self._layout_pass(brief)

        # Pass 2 + 3 in parallel (standard model)
        slot_fills, design_tokens = await asyncio.gather(
            self._content_pass(brief, template_selection),
            self._design_pass(brief, brand_config),
        )

        # Assemble plan
        return EmailBuildPlan(
            template=template_selection,
            slot_fills=slot_fills,
            design_tokens=design_tokens,
            ...
        )

    async def _layout_pass(self, brief: str) -> TemplateSelection:
        """LLM picks template + sections. Structured output."""

    async def _content_pass(self, brief: str, template: TemplateSelection) -> tuple[SlotFill, ...]:
        """LLM fills each slot. Per-slot retry on length/quality failure."""

    async def _design_pass(self, brief: str, brand: dict | None) -> DesignTokens:
        """LLM picks colours, fonts, spacing. Structured output."""
```

#### Step 3: Wire into ScaffolderService.process()

```python
# In scaffolder/service.py
async def process(self, request: ScaffolderRequest) -> ScaffolderResponse:
    pipeline = ScaffolderPipeline(self._registry, self._provider)

    # Phase 1: LLM decisions (structured JSON)
    plan = await pipeline.execute(request.brief, request.brand_config)

    # Phase 2: Deterministic assembly (no LLM)
    assembler = TemplateAssembler(self._registry)
    html = assembler.assemble(plan)

    # Phase 3: Auto-repair (deterministic, no LLM) — see 11.22.4
    html = await self._repair_pipeline.run(html)

    # Phase 4: QA gate
    qa_results = await self._run_qa(html)

    return ScaffolderResponse(html=html, plan=plan, ...)
```

#### Step 1: Define repair stage protocol

```python
# app/ai/agents/repair_pipeline.py
from __future__ import annotations
from dataclasses import dataclass, field
from typing import Protocol

@dataclass
class RepairResult:
    html: str
    repairs_applied: list[str] = field(default_factory=list)
    warnings: list[str] = field(default_factory=list)

class RepairStage(Protocol):
    """A single deterministic repair stage. No LLM calls."""

    @property
    def name(self) -> str: ...

    def repair(self, html: str) -> RepairResult: ...


class RepairPipeline:
    """
    Cascading auto-repair: runs 7 stages in sequence.
    Each stage receives the output of the previous.
    All stages are deterministic — no LLM.
    """

    def __init__(self, stages: list[RepairStage] | None = None) -> None:
        self._stages = stages or self._default_stages()

    def run(self, html: str) -> RepairResult:
        """Run all stages sequentially. Return final HTML + all repairs."""
        all_repairs: list[str] = []
        all_warnings: list[str] = []
        current = html

        for stage in self._stages:
            result = stage.repair(current)
            current = result.html
            all_repairs.extend(result.repairs_applied)
            all_warnings.extend(result.warnings)

        return RepairResult(html=current, repairs_applied=all_repairs, warnings=all_warnings)

    @staticmethod
    def _default_stages() -> list[RepairStage]:
        from .repair_stages.structure import StructureRepair
        from .repair_stages.mso import MSORepair
        from .repair_stages.dark_mode import DarkModeRepair
        from .repair_stages.accessibility import AccessibilityRepair
        from .repair_stages.personalisation import PersonalisationRepair
        from .repair_stages.size import SizeRepair
        from .repair_stages.links import LinkRepair
        return [
            StructureRepair(),     # 1. Fix skeleton issues
            MSORepair(),           # 2. Balance MSO conditionals, add namespaces
            DarkModeRepair(),      # 3. Inject meta tags, ensure media query exists
            AccessibilityRepair(), # 4. Add lang, role, scope, fix decorative images
            PersonalisationRepair(), # 5. Balance ESP tag delimiters
            SizeRepair(),          # 6. Strip unnecessary whitespace/comments
            LinkRepair(),          # 7. Fix empty hrefs, validate URL format
        ]
```

#### Step 2: Implement each repair stage

Each stage wraps or extends existing validators:

- **StructureRepair**: Ensure `<!DOCTYPE>`, `<html>`, `<head>`, `<body>` exist; close unclosed tags via lxml parser round-trip
- **MSORepair**: Delegate to existing `mso_repair.repair_mso_issues()` + add namespace injection
- **DarkModeRepair**: Delegate to existing `meta_injector.inject_missing_meta_tags()` + ensure `@media (prefers-color-scheme: dark)` block exists
- **AccessibilityRepair**: Programmatically add `lang="en"` if missing, `role="presentation"` on layout tables, `scope="col"` on `<th>`, `alt=""` on decorative images
- **PersonalisationRepair**: Balance unclosed Liquid/AMPscript/HubL delimiters (count openers vs closers)
- **SizeRepair**: Strip HTML comments (except MSO conditionals), collapse whitespace, remove empty `style=""` attributes
- **LinkRepair**: Replace empty `href=""` with `href="#"`, validate `href` format via `urlparse`

```python
def summarise_handoffs(history: list[AgentHandoff], max_tokens: int = 1000) -> str:
    """Compact summary of prior agent decisions for retry context."""
    lines = []
    for h in history:
        lines.append(f"- {h.agent_name}: {', '.join(h.decisions[:3])}")
        if h.warnings:
            lines.append(f"  warnings: {', '.join(h.warnings[:3])}")
    return truncate_to_tokens("\n".join(lines), max_tokens)
```
