# Plan: 11.22 — Template-First Hybrid Architecture

## Problem Statement

Hub agents currently have a **16.7% eval pass rate** (baseline 2026-03-09). Root cause: agents generate raw HTML via unconstrained LLM output, where every token is a chance for non-deterministic structural failures. MSO conditionals pass at 0%, accessibility at 8%, HTML preservation at 10%.

The PIV loop pattern (Plan → Implement → Validate) from the deterministic agentic coding workshop and Stripe's Minions architecture both demonstrate the same solution: **LLMs make decisions in structured JSON; deterministic code assembles the output**.

## Architecture Change

```
CURRENT (per agent):
  Brief → LLM generates raw HTML → per-agent post-process → QA gate

TARGET (per agent):
  Brief → LLM returns structured decisions (JSON) → deterministic assembler → cascading auto-repair → QA gate
```

The LLM becomes an **architect** (deciding what to do) rather than a **builder** (writing HTML). Deterministic code handles assembly, repair, and validation.

## Key Decisions

| # | Decision | Rationale |
|---|----------|-----------|
| 1 | Plan all 9 subtasks, execute incrementally starting W1 | Full picture upfront |
| 2 | Golden templates: Maizzle source + pre-compiled HTML | Fast assembly + Maizzle ecosystem |
| 3 | Provider-agnostic structured output | Fits existing adapter pattern; Anthropic→tool_use, OpenAI→response_format |
| 4 | Repair pipeline in `app/qa_engine/repair/` | Pairs with QA checks, reuses domain logic |
| 5 | Validate all 11 QA checks with sensible defaults | Production-ready templates |
| 6 | Reuse Maizzle components as section block starting points | Avoid reinvention, harden for QA compliance |

## Success Criteria

- Scaffolder structural pass rate: 0% → 95%+
- Overall eval pass rate: 16.7% → 85%+ (M1), then 95%+ (M2), then 99%+ (M4)
- Zero agent test regressions (all existing tests pass)
- Type checking clean (mypy + pyright 0 errors)
- `make check` passes

```python
from dataclasses import dataclass, field
from typing import Literal

SlotType = Literal[
    "headline", "subheadline", "body", "cta", "image",
    "preheader", "footer", "nav", "social", "divider",
]

LayoutType = Literal[
    "newsletter", "promotional", "transactional",
    "event", "retention", "announcement", "minimal",
]

@dataclass(frozen=True)
class TemplateSlot:
    """A content slot within a golden template."""
    slot_id: str                    # e.g., "hero_headline", "body_1", "cta_primary"
    slot_type: SlotType             # semantic type
    selector: str                   # CSS selector to locate in HTML (e.g., "[data-slot='hero_headline']")
    required: bool = True           # must be filled
    max_chars: int | None = None    # optional length constraint
    placeholder: str = ""           # default placeholder text in template

@dataclass(frozen=True)
class TemplateMetadata:
    """Metadata for template selection by the LLM."""
    name: str                       # e.g., "newsletter_1col"
    display_name: str               # e.g., "Single-Column Newsletter"
    layout_type: LayoutType
    column_count: int               # 1, 2, 3
    has_hero_image: bool
    has_navigation: bool
    has_social_links: bool
    sections: tuple[str, ...]       # ordered section names
    ideal_for: tuple[str, ...]      # e.g., ("weekly digest", "blog roundup")
    description: str                # 1-2 sentence for LLM selection

@dataclass(frozen=True)
class GoldenTemplate:
    """A pre-validated email template skeleton."""
    metadata: TemplateMetadata
    html: str                       # Full compiled HTML with slot placeholders
    slots: tuple[TemplateSlot, ...]
    maizzle_source: str = ""        # Original Maizzle source (for recompilation)
```

**Key requirements for ALL golden templates:**
1. Use `data-slot="{slot_id}"` attributes to mark content slots
2. Extend `src/layouts/main.html` (inherits MSO skeleton, dark mode meta, namespaces)
3. Use `<component>` includes for reusable parts (header, CTA, footer)
4. All layout tables must have `role="presentation"`
5. All images must have `alt` attributes (even if empty for decorative)
6. All `<th>` elements must have `scope` attribute
7. Use inline styles (Maizzle inlines CSS during build)
8. Placeholder content in every slot

#### Step 4: Build template compiler

`app/ai/templates/compiler.py`:

```python
"""Compile Maizzle source templates to production HTML.

Uses the maizzle-builder sidecar (services/maizzle-builder) to compile
templates. Caches compiled HTML in library/ directory.
"""
import httpx
from pathlib import Path
from app.core.logging import get_logger

logger = get_logger(__name__)

MAIZZLE_BUILDER_URL = "http://localhost:3001"
MAIZZLE_SRC_DIR = Path(__file__).parent / "maizzle_src"
LIBRARY_DIR = Path(__file__).parent / "library"

async def compile_template(name: str) -> str:
    """Compile a single Maizzle source template to production HTML.

    Args:
        name: Template name (without .html extension).

    Returns:
        Compiled HTML string.
    """
    source_path = MAIZZLE_SRC_DIR / f"{name}.html"
    source = source_path.read_text()

    async with httpx.AsyncClient(timeout=30.0) as client:
        response = await client.post(
            f"{MAIZZLE_BUILDER_URL}/build",
            json={"source": source, "options": {"inlineCSS": True}},
        )
        response.raise_for_status()
        compiled = response.json()["html"]

    # Cache compiled HTML
    output_path = LIBRARY_DIR / f"{name}.html"
    output_path.write_text(compiled)
    logger.info("templates.compile_completed", template=name, size=len(compiled))
    return compiled

async def compile_all() -> dict[str, str]:
    """Compile all Maizzle source templates. Returns {name: html}."""
    results: dict[str, str] = {}
    for source_file in sorted(MAIZZLE_SRC_DIR.glob("*.html")):
        name = source_file.stem
        results[name] = await compile_template(name)
    return results
```

**IMPORTANT**: Compiled HTML is also committed to `library/` so that templates work without the Maizzle sidecar running. The compiler is used for regeneration/updates only.

```python
"""Template registry — loads, indexes, and serves golden templates."""
from __future__ import annotations
import re
import yaml
from pathlib import Path
from typing import Literal

from app.ai.templates.models import GoldenTemplate, LayoutType, TemplateMetadata, TemplateSlot
from app.core.logging import get_logger

logger = get_logger(__name__)

LIBRARY_DIR = Path(__file__).parent / "library"
METADATA_DIR = LIBRARY_DIR / "_metadata"
MAIZZLE_SRC_DIR = Path(__file__).parent / "maizzle_src"

# Slot marker pattern: data-slot="slot_id" on any element
_SLOT_PATTERN = re.compile(r'data-slot=["\']([^"\']+)["\']')


class TemplateRegistry:
    """Loads, indexes, and serves golden templates."""

    def __init__(self) -> None:
        self._templates: dict[str, GoldenTemplate] = {}
        self._loaded = False

    def load(self) -> None:
        """Load all templates from library/ directory."""
        if self._loaded:
            return

        for html_file in sorted(LIBRARY_DIR.glob("*.html")):
            name = html_file.stem
            metadata_file = METADATA_DIR / f"{name}.yaml"
            if not metadata_file.exists():
                logger.warning("templates.missing_metadata", template=name)
                continue

            html = html_file.read_text()
            metadata_raw = yaml.safe_load(metadata_file.read_text())
            metadata = _parse_metadata(metadata_raw)
            slots = _parse_slots(metadata_raw.get("slots", []))

            # Load Maizzle source if available
            maizzle_src = ""
            src_file = MAIZZLE_SRC_DIR / f"{name}.html"
            if src_file.exists():
                maizzle_src = src_file.read_text()

            self._templates[name] = GoldenTemplate(
                metadata=metadata,
                html=html,
                slots=slots,
                maizzle_source=maizzle_src,
            )

        self._loaded = True
        logger.info("templates.registry_loaded", count=len(self._templates))

    def get(self, name: str) -> GoldenTemplate | None:
        """Get template by name."""
        self._ensure_loaded()
        return self._templates.get(name)

    def search(
        self,
        layout_type: LayoutType | None = None,
        column_count: int | None = None,
        has_hero: bool | None = None,
    ) -> list[GoldenTemplate]:
        """Filter templates by criteria."""
        self._ensure_loaded()
        results = list(self._templates.values())
        if layout_type is not None:
            results = [t for t in results if t.metadata.layout_type == layout_type]
        if column_count is not None:
            results = [t for t in results if t.metadata.column_count == column_count]
        if has_hero is not None:
            results = [t for t in results if t.metadata.has_hero_image == has_hero]
        return results

    def list_for_selection(self) -> list[TemplateMetadata]:
        """Return metadata-only list for LLM template selection prompt."""
        self._ensure_loaded()
        return [t.metadata for t in self._templates.values()]

    def fill_slots(self, template: GoldenTemplate, fills: dict[str, str]) -> str:
        """Replace slot placeholder content with fills. Deterministic string replacement.

        Uses data-slot attributes to locate slots in the HTML DOM.
        Replaces the inner content of elements that have data-slot="X" with fills["X"].
        """
        html = template.html
        for slot in template.slots:
            if slot.slot_id not in fills:
                continue
            fill_content = fills[slot.slot_id]
            # Enforce max_chars if set
            if slot.max_chars and len(fill_content) > slot.max_chars:
                fill_content = fill_content[:slot.max_chars]
            # Replace content within data-slot element
            # Pattern: <tag data-slot="slot_id" ...>placeholder</tag>
            pattern = re.compile(
                rf'(<[^>]+data-slot=["\']' + re.escape(slot.slot_id) + r'["\'][^>]*>)(.*?)(</[^>]+>)',
                re.DOTALL,
            )
            html = pattern.sub(rf'\1{re.escape(fill_content)}\3', html, count=1)
        return html

    def names(self) -> list[str]:
        """Return all template names."""
        self._ensure_loaded()
        return list(self._templates.keys())

    def _ensure_loaded(self) -> None:
        if not self._loaded:
            self.load()


def _parse_metadata(raw: dict[str, object]) -> TemplateMetadata:
    """Parse YAML metadata dict into TemplateMetadata."""
    return TemplateMetadata(
        name=str(raw["name"]),
        display_name=str(raw["display_name"]),
        layout_type=str(raw["layout_type"]),  # type: ignore[arg-type]
        column_count=int(raw.get("column_count", 1)),  # type: ignore[arg-type]
        has_hero_image=bool(raw.get("has_hero_image", False)),
        has_navigation=bool(raw.get("has_navigation", False)),
        has_social_links=bool(raw.get("has_social_links", False)),
        sections=tuple(raw.get("sections", [])),  # type: ignore[arg-type]
        ideal_for=tuple(raw.get("ideal_for", [])),  # type: ignore[arg-type]
        description=str(raw.get("description", "")),
    )


def _parse_slots(raw_slots: list[dict[str, object]]) -> tuple[TemplateSlot, ...]:
    """Parse YAML slot definitions into TemplateSlot tuple."""
    slots = []
    for s in raw_slots:
        slots.append(TemplateSlot(
            slot_id=str(s["slot_id"]),
            slot_type=str(s["slot_type"]),  # type: ignore[arg-type]
            selector=str(s["selector"]),
            required=bool(s.get("required", True)),
            max_chars=int(s["max_chars"]) if s.get("max_chars") else None,
            placeholder=str(s.get("placeholder", "")),
        ))
    return tuple(slots)


# Module-level singleton
_registry: TemplateRegistry | None = None

def get_template_registry() -> TemplateRegistry:
    """Get or create the global template registry singleton."""
    global _registry
    if _registry is None:
        _registry = TemplateRegistry()
    return _registry
```

```python
"""Deterministic HTML assembly from structured plan + golden template.

This module contains ZERO LLM calls. It takes an EmailBuildPlan and
a GoldenTemplate and produces final HTML through string operations.
"""
import re
from app.ai.agents.schemas.build_plan import EmailBuildPlan, DesignTokens, SectionDecision
from app.ai.templates import GoldenTemplate, TemplateRegistry, get_template_registry
from app.core.logging import get_logger

logger = get_logger(__name__)


class TemplateAssembler:
    """Deterministic HTML assembly from structured plan + golden template."""

    def __init__(self, registry: TemplateRegistry | None = None) -> None:
        self._registry = registry or get_template_registry()

    def assemble(self, plan: EmailBuildPlan) -> str:
        """Assemble final HTML from plan. 100% deterministic.

        1. Load golden template by plan.template.template_name
        2. Fill slots with plan.slot_fills
        3. Apply design tokens (inline style injection)
        4. Apply section decisions (show/hide, background overrides)
        5. Set preheader text
        6. Return complete HTML
        """
        template = self._registry.get(plan.template.template_name)
        if template is None:
            # Fallback to fallback_template
            if plan.template.fallback_template:
                template = self._registry.get(plan.template.fallback_template)
            if template is None:
                raise ValueError(
                    f"Template '{plan.template.template_name}' not found in registry. "
                    f"Available: {self._registry.names()}"
                )

        # Step 1: Fill slots
        fills = {sf.slot_id: sf.content for sf in plan.slot_fills}
        html = self._registry.fill_slots(template, fills)

        # Step 2: Apply design tokens
        html = self._apply_design_tokens(html, plan.design_tokens)

        # Step 3: Apply section decisions
        html = self._apply_sections(html, plan.sections)

        # Step 4: Set preheader
        if plan.preheader_text:
            html = self._set_preheader(html, plan.preheader_text)

        logger.info(
            "scaffolder.assembly_completed",
            template=plan.template.template_name,
            slots_filled=len(fills),
            html_length=len(html),
        )
        return html

    def _apply_design_tokens(self, html: str, tokens: DesignTokens) -> str:
        """Inject design token values into CSS custom properties and inline styles."""
        # Replace CSS custom property defaults if they exist
        replacements = {
            "--color-primary": tokens.primary_color,
            "--color-secondary": tokens.secondary_color,
            "--color-bg": tokens.background_color,
            "--color-text": tokens.text_color,
            "--font-body": tokens.font_family,
            "--font-heading": tokens.heading_font_family,
            "--border-radius": tokens.border_radius,
        }
        for prop, value in replacements.items():
            html = re.sub(
                rf'var\({re.escape(prop)},\s*[^)]+\)',
                value,
                html,
            )
        return html

    def _apply_sections(self, html: str, sections: tuple[SectionDecision, ...]) -> str:
        """Show/hide sections, apply background overrides."""
        for section in sections:
            if section.hidden:
                # Remove section by data-section attribute
                html = re.sub(
                    rf'<[^>]+data-section=["\']' + re.escape(section.section_name) + r'["\'][^>]*>.*?</[^>]+>',
                    '',
                    html,
                    flags=re.DOTALL,
                    count=1,
                )
            elif section.background_color:
                # Override section background
                pattern = re.compile(
                    rf'(<[^>]+data-section=["\']' + re.escape(section.section_name) + r'["\'][^>]*style=["\'])([^"\']*)',
                )
                html = pattern.sub(
                    rf'\1background-color:{section.background_color};\2',
                    html,
                    count=1,
                )
        return html

    def _set_preheader(self, html: str, preheader: str) -> str:
        """Set preheader text in the preheader slot."""
        # Use data-slot="preheader" if it exists
        if 'data-slot="preheader"' in html:
            pattern = re.compile(
                r'(<[^>]+data-slot=["\']preheader["\'][^>]*>)(.*?)(</[^>]+>)',
                re.DOTALL,
            )
            return pattern.sub(rf'\1{re.escape(preheader)}\3', html, count=1)
        return html
```

```python
"""3-pass generation pipeline for the Scaffolder agent.

Pass 1 (Layout):  Brief → TemplateSelection + SectionDecisions
Pass 2 (Content): Brief + Template → SlotFills
Pass 3 (Design):  Brief + Brand → DesignTokens

Each pass uses structured output. Passes 2+3 run in parallel.
Per-slot retry on content failures.
"""
import asyncio
import json
from typing import Any

from app.ai.agents.schemas.build_plan import (
    DesignTokens,
    EmailBuildPlan,
    SectionDecision,
    SlotFill,
    TemplateSelection,
)
from app.ai.protocols import CompletionResponse, LLMProvider, Message
from app.ai.templates import TemplateRegistry, get_template_registry
from app.core.logging import get_logger

logger = get_logger(__name__)


class ScaffolderPipeline:
    """3-pass generation pipeline with structured output."""

    def __init__(
        self,
        provider: LLMProvider,
        model: str,
        registry: TemplateRegistry | None = None,
        max_tokens: int = 4096,
    ) -> None:
        self._provider = provider
        self._model = model
        self._registry = registry or get_template_registry()
        self._max_tokens = max_tokens

    async def execute(
        self,
        brief: str,
        brand_config: dict[str, object] | None = None,
    ) -> EmailBuildPlan:
        """Execute 3-pass pipeline. Returns complete EmailBuildPlan."""
        # Pass 1: Layout selection (lightweight — fast)
        template_selection, section_decisions = await self._layout_pass(brief)

        # Load selected template to inform content pass
        template = self._registry.get(template_selection.template_name)
        slot_ids = [s.slot_id for s in template.slots] if template else []

        # Pass 2 + 3 in parallel
        slot_fills, design_tokens = await asyncio.gather(
            self._content_pass(brief, template_selection, slot_ids),
            self._design_pass(brief, brand_config),
        )

        plan = EmailBuildPlan(
            template=template_selection,
            slot_fills=slot_fills,
            design_tokens=design_tokens,
            sections=section_decisions,
            confidence=0.85,  # Base confidence, refined by QA
            reasoning=template_selection.reasoning,
        )
        logger.info(
            "scaffolder.pipeline_completed",
            template=template_selection.template_name,
            slots=len(slot_fills),
        )
        return plan

    async def _layout_pass(
        self, brief: str
    ) -> tuple[TemplateSelection, tuple[SectionDecision, ...]]:
        """Pass 1: LLM picks template + sections. Structured output."""
        available = self._registry.list_for_selection()
        template_list = "\n".join(
            f"- {m.name}: {m.display_name} ({m.layout_type}, {m.column_count}-col) — {m.description}"
            for m in available
        )

        system = (
            "You are an email layout architect. Select the best template for the brief.\n"
            "Return JSON with: template_name, reasoning, section_order (if __compose__), "
            "and section_decisions (list of {section_name, background_color?, hidden?}).\n"
            f"\nAvailable templates:\n{template_list}\n"
            "\nIf no template matches well, use template_name='__compose__' and specify "
            "section_order from available section blocks."
        )
        user = f"Campaign brief:\n{brief}"

        result = await self._provider.complete(
            [Message(role="system", content=system), Message(role="user", content=user)],
            model_override=self._model,
            max_tokens=self._max_tokens,
        )

        parsed = self._parse_json(result.content)
        template_selection = TemplateSelection(
            template_name=str(parsed.get("template_name", "")),
            reasoning=str(parsed.get("reasoning", "")),
            section_order=tuple(parsed.get("section_order", [])),
            fallback_template=parsed.get("fallback_template"),
        )
        section_decisions = tuple(
            SectionDecision(
                section_name=str(s.get("section_name", "")),
                background_color=s.get("background_color"),
                padding=s.get("padding"),
                hidden=bool(s.get("hidden", False)),
            )
            for s in parsed.get("section_decisions", [])
        )
        return template_selection, section_decisions

    async def _content_pass(
        self,
        brief: str,
        template: TemplateSelection,
        slot_ids: list[str],
    ) -> tuple[SlotFill, ...]:
        """Pass 2: LLM fills each slot. Per-slot retry on failure."""
        system = (
            "You are an email content writer. Fill each content slot with appropriate HTML content.\n"
            "Return JSON with 'slot_fills': list of {slot_id, content, is_personalisable}.\n"
            f"Slots to fill: {slot_ids}\n"
            "Content should be concise, professional, and match the campaign brief."
        )
        user = f"Template: {template.template_name}\nBrief:\n{brief}"

        result = await self._provider.complete(
            [Message(role="system", content=system), Message(role="user", content=user)],
            model_override=self._model,
            max_tokens=self._max_tokens,
        )

        parsed = self._parse_json(result.content)
        return tuple(
            SlotFill(
                slot_id=str(sf.get("slot_id", "")),
                content=str(sf.get("content", "")),
                is_personalisable=bool(sf.get("is_personalisable", False)),
            )
            for sf in parsed.get("slot_fills", [])
        )

    async def _design_pass(
        self,
        brief: str,
        brand_config: dict[str, object] | None = None,
    ) -> DesignTokens:
        """Pass 3: LLM picks colours, fonts, spacing. Structured output."""
        brand_context = ""
        if brand_config:
            brand_context = f"\nBrand guidelines: {json.dumps(brand_config, default=str)}"

        system = (
            "You are an email design architect. Choose design tokens for the email.\n"
            "Return JSON with: primary_color, secondary_color, background_color, text_color, "
            "font_family, heading_font_family, border_radius, button_style.\n"
            "Use web-safe hex colors and font stacks."
            f"{brand_context}"
        )
        user = f"Brief:\n{brief}"

        result = await self._provider.complete(
            [Message(role="system", content=system), Message(role="user", content=user)],
            model_override=self._model,
            max_tokens=1024,
        )

        parsed = self._parse_json(result.content)
        return DesignTokens(
            primary_color=str(parsed.get("primary_color", "#e84e0f")),
            secondary_color=str(parsed.get("secondary_color", "#0c2340")),
            background_color=str(parsed.get("background_color", "#ffffff")),
            text_color=str(parsed.get("text_color", "#333333")),
            font_family=str(parsed.get("font_family", "-apple-system,BlinkMacSystemFont,'Segoe UI',Roboto,sans-serif")),
            heading_font_family=str(parsed.get("heading_font_family", "Georgia,serif")),
            border_radius=str(parsed.get("border_radius", "4px")),
            button_style=parsed.get("button_style", "filled"),
        )

    @staticmethod
    def _parse_json(content: str) -> dict[str, Any]:
        """Extract JSON from LLM response (may be wrapped in code fence)."""
        import re
        # Try to find JSON in code fence
        match = re.search(r'```(?:json)?\s*\n?(.*?)\n?```', content, re.DOTALL)
        if match:
            return json.loads(match.group(1))
        # Try raw JSON parse
        try:
            return json.loads(content)
        except json.JSONDecodeError:
            logger.warning("scaffolder.json_parse_failed", content_preview=content[:200])
            return {}
```

### Implementation Steps

#### Step 1: Define repair stage protocol and pipeline

`app/qa_engine/repair/__init__.py`:

`app/qa_engine/repair/pipeline.py`:

```python
"""Cascading auto-repair pipeline — 7 deterministic stages, zero LLM calls.

Runs before the QA gate to fix known structural issues. Each stage is
a pure function: HTML in → (repaired HTML, repairs list, warnings list) out.
"""
from __future__ import annotations
from dataclasses import dataclass, field
from typing import Protocol

from app.core.logging import get_logger

logger = get_logger(__name__)


@dataclass
class RepairResult:
    html: str
    repairs_applied: list[str] = field(default_factory=list)
    warnings: list[str] = field(default_factory=list)


class RepairStage(Protocol):
    """A single deterministic repair stage. No LLM calls."""

    @property
    def name(self) -> str: ...

    def repair(self, html: str) -> RepairResult: ...


class RepairPipeline:
    """Cascading auto-repair: runs 7 stages in sequence.

    Each stage receives the output of the previous.
    All stages are deterministic — no LLM.
    """

    def __init__(self, stages: list[RepairStage] | None = None) -> None:
        self._stages = stages if stages is not None else self._default_stages()

    def run(self, html: str) -> RepairResult:
        """Run all stages sequentially. Return final HTML + all repairs."""
        all_repairs: list[str] = []
        all_warnings: list[str] = []
        current = html

        for stage in self._stages:
            try:
                result = stage.repair(current)
                if result.repairs_applied:
                    logger.info(
                        f"repair.{stage.name}_applied",
                        count=len(result.repairs_applied),
                    )
                current = result.html
                all_repairs.extend(result.repairs_applied)
                all_warnings.extend(result.warnings)
            except Exception as e:
                logger.warning(
                    f"repair.{stage.name}_failed",
                    error=str(e),
                )
                all_warnings.append(f"{stage.name}: repair failed ({e})")

        return RepairResult(
            html=current,
            repairs_applied=all_repairs,
            warnings=all_warnings,
        )

    @staticmethod
    def _default_stages() -> list[RepairStage]:
        from app.qa_engine.repair.structure import StructureRepair
        from app.qa_engine.repair.mso import MSORepair
        from app.qa_engine.repair.dark_mode import DarkModeRepair
        from app.qa_engine.repair.accessibility import AccessibilityRepair
        from app.qa_engine.repair.personalisation import PersonalisationRepair
        from app.qa_engine.repair.size import SizeRepair
        from app.qa_engine.repair.links import LinkRepair
        return [
            StructureRepair(),        # 1. Fix HTML skeleton issues
            MSORepair(),              # 2. Balance MSO conditionals, add namespaces
            DarkModeRepair(),         # 3. Inject meta tags, ensure media query
            AccessibilityRepair(),    # 4. Add lang, role, scope, fix decorative imgs
            PersonalisationRepair(),  # 5. Balance ESP tag delimiters
            SizeRepair(),             # 6. Strip unnecessary whitespace/comments
            LinkRepair(),             # 7. Fix empty hrefs, validate URL format
        ]
```

#### Step 2: Implement each repair stage

Each stage wraps or extends existing validators:

**`structure.py`**: Ensure `<!DOCTYPE>`, `<html>`, `<head>`, `<body>` exist. Use lxml to parse + serialize (auto-closes unclosed tags). Preserve MSO comments and `data-slot` attributes.

**`mso.py`**: Delegate to existing `app/ai/agents/outlook_fixer/mso_repair.repair_mso_issues()`. Add namespace injection (`xmlns:v`, `xmlns:o`) if missing from `<html>` tag.

**`dark_mode.py`**: Delegate to existing `app/ai/agents/dark_mode/meta_injector.inject_missing_meta_tags()`. Also ensure `@media (prefers-color-scheme: dark)` block exists in `<style>` (inject empty one if missing).

**`accessibility.py`**: Programmatically add `lang="en"` if missing from `<html>`, `role="presentation"` on layout tables (tables without `data-slot`), `scope="col"` on `<th>` elements, `alt=""` on images missing alt attribute.

**`personalisation.py`**: Count opened vs closed ESP delimiters for common platforms (Liquid `{{`/`}}`, AMPscript `%%`/`%%`, Handlebars `{{`/`}}`). Warn on imbalance (don't auto-close — too risky).

**`size.py`**: Strip HTML comments (EXCEPT MSO conditionals `<!--[if`/`<![endif]-->`), collapse consecutive whitespace in text nodes, remove empty `style=""` attributes.

**`links.py`**: Replace empty `href=""` with `href="#"`, warn on `javascript:` hrefs, validate `href` format via `urllib.parse.urlparse`.

In `app/ai/blueprints/engine.py` — after agentic node returns HTML, before QA gate:
