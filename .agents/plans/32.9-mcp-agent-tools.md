## Agent Tool Response Formatting

Each agent returns different response shapes. Use `to_dict()` for Pydantic â†’ dict, then `format_simple_result()` for LLM-friendly markdown. For HTML-heavy responses (scaffolder, dark mode, outlook fixer, accessibility, personalisation), truncate HTML via `truncate_html()` and include metadata (confidence, warnings, QA status).

**Custom formatter** `format_agent_result()` in `agents.py`:

```python
def _format_agent_result(response: dict[str, Any], agent_label: str) -> str:
    lines = [f"**{agent_label}:**"]
    # Confidence first
    if conf := response.get("confidence"):
        lines.append(f"  - **confidence:** {conf:.0%}")
    # Truncate HTML if present
    if html := response.get("html"):
        response["html"] = truncate_html(html)
    # All other fields
    for k, v in response.items():
        if k not in ("html", "confidence"):
            lines.append(f"  - **{k}:** {v}")
    if html:
        lines.append(f"\n**HTML** ({len(html):,} chars):\n{response['html']}")
    return _apply_token_budget("\n".join(lines))
```
