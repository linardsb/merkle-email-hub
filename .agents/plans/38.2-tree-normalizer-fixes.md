## Implementation Steps

### Bug 6 (HIGH): Fix `item_spacing` in `_infer_auto_layout` — `tree_normalizer.py:196-208`

**Current:** Uses position differences `sorted_ys[i+1] - sorted_ys[i]` which includes child height.
**Fix:** Compute gap between bounding box edges, clamp ≥ 0, use median.

```python
# VERTICAL case (~line 196-200):
# Replace:
#   sorted_ys = sorted(ys)
#   spacings = [sorted_ys[i + 1] - sorted_ys[i] for i in range(len(sorted_ys) - 1)]
#   avg_spacing = sum(spacings) / len(spacings) if spacings else 0
# With:
sorted_children = sorted(
    [(c.y or 0.0, c.height or 0.0) for c in node.children],
    key=lambda t: t[0],
)
gaps = [
    max(0.0, sorted_children[i + 1][0] - (sorted_children[i][0] + sorted_children[i][1]))
    for i in range(len(sorted_children) - 1)
]
med_spacing = statistics.median(gaps) if gaps else 0.0

# HORIZONTAL case (~line 203-207): same pattern with x/width
sorted_children = sorted(
    [(c.x or 0.0, c.width or 0.0) for c in node.children],
    key=lambda t: t[0],
)
gaps = [
    max(0.0, sorted_children[i + 1][0] - (sorted_children[i][0] + sorted_children[i][1]))
    for i in range(len(sorted_children) - 1)
]
med_spacing = statistics.median(gaps) if gaps else 0.0
```

Add `import statistics` at top if not already present. Use `median` instead of `mean` for robustness.

### Bug 7 (MEDIUM): Remove `raw_file_data` guard — `tree_normalizer.py:92-93`

**Current:** `_resolve_instances` gated on `if raw_file_data:` but never uses it.
**Fix:** Remove the guard, resolve INSTANCE→FRAME unconditionally.


Also remove `raw_file_data` parameter from `_normalize_page()` signature and update the caller in `normalize_tree()`.

### Bug 8 (MEDIUM): Inherit width/height in group flattening — `tree_normalizer.py:139-143`

**Current:** Only inherits `x`, `y` when child has `None`.
**Fix:** Also inherit `width`, `height`.

```python
# After the x/y inheritance block, add:
if grandchild.width is None and child.width is not None:
    grandchild = replace(grandchild, width=child.width)
if grandchild.height is None and child.height is not None:
    grandchild = replace(grandchild, height=child.height)
```

### Bug 9 (MEDIUM): Fix text merge spacing estimate — `tree_normalizer.py:249-252`

**Current:** `expected_y = current.y + current.line_height_px * len(group)` — uses CSS line_height.
**Fix:** Use actual node height (bounding box) as spacing estimate.

```python
# Replace the spacing check block with:
last = group[-1]
if nxt.y is not None and last.y is not None:
    last_bottom = last.y + (last.height or last.line_height_px or 0.0)
    if abs(nxt.y - last_bottom) > _POS_TOLERANCE:
        break
```

This uses the last node in the group (not the first) and uses `height` with `line_height_px` as fallback.

### Bug 10 (LOW): Prune empty containers — `tree_normalizer.py:104-119`

**Current:** `_remove_invisible` removes invisible nodes but leaves empty parent containers.
**Fix:** After recursing into children, check if container is now empty and has no visual content.

```python
# In _remove_invisible, after recursing into child, before appending:
recursed = _remove_invisible(child, ctx)
# Add empty container check:
if (
    not recursed.children
    and recursed.type in {DesignNodeType.FRAME, DesignNodeType.GROUP}
    and not recursed.fill_color
    and not recursed.image_ref
):
    ctx.nodes_removed += 1
    changed = True
    continue
```

Containers with `fill_color` or `image_ref` are preserved (they have visual content even without children).
