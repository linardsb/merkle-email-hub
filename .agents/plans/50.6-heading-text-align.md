### Modify: `app/design_sync/component_matcher.py:1401-1469`

Add 2 emissions after the existing color block:

```python
# Text-align overrides from text-node attr (Gap 11, Phase 50.6)
for text in section.texts:
    if text.is_heading and text.text_align in ("left", "center", "right", "justify"):
        overrides.append(TokenOverride("text-align", "_heading", text.text_align.lower()))
        break

for text in section.texts:
    if not text.is_heading and text.text_align in ("left", "center", "right", "justify"):
        overrides.append(TokenOverride("text-align", "_body", text.text_align.lower()))
        break
```

That's it. The renderer's existing class-fallback machinery (Phase 49.4) handles `_heading` / `_body` targets.
