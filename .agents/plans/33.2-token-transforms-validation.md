# Plan: 33.2 Email-Safe Token Transforms & Validation Layer

## Context

Zero validation exists between token extraction (Phase 33.1 — `figma/service.py`, `penpot/service.py`) and downstream consumers (`converter.py`, `converter_service.py`, Scaffolder pipeline). Problems today:
- Empty font family → `", Arial, Helvetica, sans-serif"` in output (`converter.py:171` `_font_stack`)
- 3-digit hex or named colors could slip through if a provider returns them
- No bounds checking on sizes — a 0px font or 9999px spacing silently propagates
- Opacity already composited at extraction (`_rgba_to_hex_with_opacity` in `figma/service.py:76`), but `ExtractedColor.opacity` field still carries the raw value with no clamp
- No duplicate detection — same color extracted twice wastes palette slots

This layer sits between extraction and `DesignConverterService.convert()` / `DesignSyncService.get_tokens()`.

#### 1c. Hex validation helpers

```python
_HEX6_RE = re.compile(r"^#[0-9A-Fa-f]{6}$")
_HEX3_RE = re.compile(r"^#[0-9A-Fa-f]{3}$")
_RGBA_RE = re.compile(
    r"^rgba?\(\s*(\d+)\s*,\s*(\d+)\s*,\s*(\d+)\s*(?:,\s*([0-9.]+))?\s*\)$", re.IGNORECASE
)
_HSL_RE = re.compile(
    r"^hsla?\(\s*(\d+)\s*,\s*(\d+)%\s*,\s*(\d+)%\s*(?:,\s*([0-9.]+))?\s*\)$", re.IGNORECASE
)


def _normalize_hex(value: str) -> str | None:
    """Try to normalize a color string to 6-digit uppercase hex. Return None if unparseable."""
    value = value.strip()

    # Already valid 6-digit hex
    if _HEX6_RE.match(value):
        return value.upper()

    # 3-digit shorthand → expand
    if _HEX3_RE.match(value):
        r, g, b = value[1], value[2], value[3]
        return f"#{r}{r}{g}{g}{b}{b}".upper()

    # Named CSS color
    lower = value.lower()
    if lower in CSS_NAMED_COLORS:
        return CSS_NAMED_COLORS[lower]

    # rgba(r, g, b[, a]) — ignore alpha, convert RGB channels
    m = _RGBA_RE.match(value)
    if m:
        r_i = min(255, max(0, int(m.group(1))))
        g_i = min(255, max(0, int(m.group(2))))
        b_i = min(255, max(0, int(m.group(3))))
        return f"#{r_i:02X}{g_i:02X}{b_i:02X}"

    # hsl(h, s%, l%) → convert to RGB → hex
    m = _HSL_RE.match(value)
    if m:
        return _hsl_to_hex(int(m.group(1)), int(m.group(2)), int(m.group(3)))

    return None


def _hsl_to_hex(h: int, s: int, l: int) -> str:
    """Convert HSL to 6-digit hex. h=0-360, s/l=0-100."""
    # Standard HSL→RGB algorithm
    s_f = s / 100.0
    l_f = l / 100.0
    c = (1 - abs(2 * l_f - 1)) * s_f
    x = c * (1 - abs((h / 60.0) % 2 - 1))
    m = l_f - c / 2
    if h < 60:
        r_f, g_f, b_f = c, x, 0.0
    elif h < 120:
        r_f, g_f, b_f = x, c, 0.0
    elif h < 180:
        r_f, g_f, b_f = 0.0, c, x
    elif h < 240:
        r_f, g_f, b_f = 0.0, x, c
    elif h < 300:
        r_f, g_f, b_f = x, 0.0, c
    else:
        r_f, g_f, b_f = c, 0.0, x
    r_i = round((r_f + m) * 255)
    g_i = round((g_f + m) * 255)
    b_i = round((b_f + m) * 255)
    return f"#{r_i:02X}{g_i:02X}{b_i:02X}"
```

#### 1d. Per-type validation functions

**`_validate_color`** — for each `ExtractedColor`:
1. Normalize `hex` via `_normalize_hex()`. If different from original → `warning` with original/fixed.
2. If `_normalize_hex()` returns `None` → `error` warning, keep original hex unchanged (don't corrupt data).
3. If `value.lower() == "transparent"` → `warning` ("fully transparent color").
4. Clamp `opacity` to `[0.0, 1.0]`. If changed → `warning`.
5. If `opacity < 0.01` → `info` ("fully transparent color").
6. Return `replace(color, hex=normalized, opacity=clamped)`.

**`_validate_typography`** — for each `ExtractedTypography`:
1. If `family` is empty or whitespace → `warning`, default to `"Arial"`.
2. If `size <= 0` → `error`. If `size > 200` → `warning` ("unusually large font size").
3. Validate `weight`: must be `"normal"`, `"bold"`, or `str(N)` where N in {100,200,...,900}. If numeric string not in set → map to nearest valid (e.g. `"350"` → `"400"`). If non-numeric non-keyword → `warning`, default to `"400"`.
4. If `line_height <= 0` → `warning`, default to `size * 1.5`.
5. If `line_height` is unitless ratio (value < 5.0 and less than `size`) → multiply by `size` to get px. Emit `info`.
6. Return replaced dataclass.

**`_validate_spacing`** — for each `ExtractedSpacing`:
1. If `value < 0` → `error` ("negative spacing").
2. If `value > 500` → `warning` ("unusually large spacing").
3. Return spacing unchanged (or clamped if negative → 0).

#### 1e. Cross-token validation

After per-token validation:
1. If `len(tokens.colors) == 0` → `warning` ("no colors extracted").
2. If `len(tokens.typography) == 0` → `warning` ("no typography styles extracted").
3. Deduplicate: within `colors`, if two entries share same `name` and `hex` → keep first, emit `info` for removed duplicates. Same for `typography` (match on `name`+`family`+`size`) and `spacing` (match on `name`+`value`).

#### 1f. Main entry point

```python
def validate_and_transform(
    tokens: ExtractedTokens,
) -> tuple[ExtractedTokens, list[TokenWarning]]:
    """Validate and transform extracted tokens to email-safe values.

    Returns a new ExtractedTokens with fixes applied, plus a list of warnings.
    """
    warnings: list[TokenWarning] = []

    # Per-token validation
    colors = [_validate_color(c, warnings) for c in tokens.colors]
    typography = [_validate_typography(t, warnings) for t in tokens.typography]
    spacing = [_validate_spacing(s, warnings) for s in tokens.spacing]

    # Deduplicate
    colors = _dedup_colors(colors, warnings)
    typography = _dedup_typography(typography, warnings)
    spacing = _dedup_spacing(spacing, warnings)

    # Cross-token warnings
    if not colors:
        warnings.append(TokenWarning(level="warning", field="colors", message="No colors extracted"))
    if not typography:
        warnings.append(
            TokenWarning(level="warning", field="typography", message="No typography styles extracted")
        )

    # Also validate stroke_colors (same rules as colors)
    stroke_colors = [_validate_color(c, warnings) for c in tokens.stroke_colors]
    stroke_colors = _dedup_colors(stroke_colors, warnings)

    # Log summary
    error_count = sum(1 for w in warnings if w.level == "error")
    warn_count = sum(1 for w in warnings if w.level == "warning")
    if error_count or warn_count:
        logger.info(
            "design_sync.token_validation",
            errors=error_count,
            warnings=warn_count,
            total_issues=len(warnings),
        )

    validated = ExtractedTokens(
        colors=colors,
        typography=typography,
        spacing=spacing,
        variables_source=tokens.variables_source,
        modes=tokens.modes,
        stroke_colors=stroke_colors,
        variables=tokens.variables,  # pass through — variables are metadata, not CSS
    )
    return validated, warnings
```
