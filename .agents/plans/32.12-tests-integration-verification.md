## Preflight Warnings

- `discover_overlays` uses `@lru_cache` — must call `.cache_clear()` in teardown/fixture to avoid cross-test contamination.
- `_AGENTS_BASE` and `_OVERLAYS_BASE` are module-level `Path` objects — patch at module level, not instance level.
- MCP tool functions are registered via decorators on `FastMCP` — access via `mcp._tool_manager._tools["tool_name"].fn` for direct invocation.
- Git operations in `skill_version.py` use `subprocess.run` — always mock to avoid hitting real git in tests.
- Existing `test_skill_versioning.py` (37 tests) is in `evals/tests/`; new file goes in `agents/tests/` for integration-level scenarios.
- Existing `test_skill_loader.py` (35 tests) covers basic overlay unit tests; new `test_skill_overlays.py` focuses on end-to-end overlay scenarios.
