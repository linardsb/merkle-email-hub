## Implementation Steps

### Step 1: Create the intelligence page

Create `cms/apps/web/src/app/[locale]/(dashboard)/intelligence/page.tsx` as a `"use client"` page.

**Imports:**
```typescript
"use client";

import { useTranslations } from "next-intl";
import { BarChart3, TrendingUp, ListChecks, Clock, MonitorSmartphone } from "lucide-react";
import { useQADashboard } from "@/hooks/use-qa-dashboard";
import { ScoreOverviewCards } from "@/components/intelligence/score-overview-cards";
import { CheckPerformanceChart } from "@/components/intelligence/check-performance-chart";
import { ScoreTrendBars } from "@/components/intelligence/score-trend-bars";
import { RecentResultsTable } from "@/components/intelligence/recent-results-table";
```

**Layout structure** (follows dashboard page pattern — `space-y-6`):
```
space-y-6
├── Header (BarChart3 icon + t("title") h1 + t("subtitle") paragraph)
├── Loading state OR content:
│   ├── ScoreOverviewCards (4-col grid, metrics prop)
│   ├── Two-column grid (gap-6 lg:grid-cols-2)
│   │   ├── Card: CheckPerformanceChart
│   │   │   ├── header: ListChecks icon + t("checkPerformance") + t("checkPerformanceDescription")
│   │   │   └── content: <CheckPerformanceChart checkAverages={metrics.checkAverages} />
│   │   └── Card: ScoreTrendBars
│   │       ├── header: TrendingUp icon + t("scoreTrend") + t("scoreTrendDescription")
│   │       └── content: <ScoreTrendBars scoreTrend={metrics.scoreTrend} />
│   ├── Card: RecentResultsTable
│   │   ├── header: Clock icon + t("recentResults") + t("recentResultsDescription")
│   │   └── content: <RecentResultsTable />
│   └── Card: Email Client Rendering placeholder
│       ├── header: MonitorSmartphone icon + t("clientRendering") + "Coming Soon" badge
│       └── content: t("clientRenderingDescription")
```

**States:**
- **Loading** (`isLoading`): Skeleton grid — 4 cards in `lg:grid-cols-4` with `h-28 animate-pulse rounded-lg bg-surface-muted`, plus 2 larger skeletons `h-64`
- **Empty** (`metrics.totalRuns === 0 && !isLoading`): Centered card with BarChart3 icon (h-12 w-12), `t("noResults")` heading, `t("noResultsDescription")` description
- **Error** (`error`): Error card with `text-status-danger` message
- **Data**: Full dashboard layout as described above

**Section card wrapper pattern** (inline, not abstracted):
```tsx
<div className="rounded-lg border border-card-border bg-card-bg p-6">
  <div className="flex items-center gap-2">
    <Icon className="h-5 w-5 text-foreground-muted" />
    <h2 className="text-lg font-semibold text-foreground">{t("sectionTitle")}</h2>
  </div>
  <p className="mt-1 text-sm text-foreground-muted">{t("sectionDescription")}</p>
  <div className="mt-4">
    {/* Component */}
  </div>
</div>
```

**"Coming Soon" badge:**
```tsx
<span className="rounded-full bg-surface-muted px-2 py-0.5 text-xs font-medium text-foreground-muted">
  {t("comingSoon")}
</span>
```

This goes in the header row of the Email Client Rendering card, using `flex items-center justify-between`.
