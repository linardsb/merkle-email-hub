# Plan: 39.6 Component Matcher Architectural Improvements

## Context

Current matcher uses first-match logic: `has_images and has_texts → article-card` at confidence 1.0. Mammut product grids get article-card treatment. Round-robin column distribution ignores spatial position. Unfilled slots show placeholder text. Need multi-candidate scoring, spatial column assignment, post-render slot validation, 3 new component types, and confidence propagation.

### Current Architecture

**`_match_content()`** (component_matcher.py:183–210): if-elif chain, returns `(slug, confidence)`. First branch `has_images and has_texts → article-card, 1.0` catches everything with mixed content.

**`_build_column_fills()`** (component_matcher.py:602–613): round-robin `i % col_count`. Uses `column_groups` if present (Phase 35), else distributes by index.

**`ComponentMatch`** (component_matcher.py:37–47): `section_idx`, `section`, `component_slug`, `slot_fills`, `token_overrides`, `spacing_after`, `confidence`, `mjml_template`.

**`ConversionResult`** (converter_service.py:130–140): `html`, `sections_count`, `warnings`, `layout`, `compatibility_hints`, `images`, `cache_hit_rate`. Missing `match_confidences`.

### Slot Fill Pattern

Each `_fills_{slug}()` returns `list[SlotFill]`. Helpers: `_first_heading()`, `_body_texts()`, `_safe_text()`, `_resolve_image_url()`. No side effects.

### Step 2: Replace `_match_content()` with `_score_candidates()`

**File:** `component_matcher.py:183–210`

New function signature:
```python

    section: EmailSection,
    has_images: bool,
    has_texts: bool,
    has_buttons: bool,
    has_headings: bool,
) -> tuple[str, float]:
```

Scoring rules (evaluated for all candidates, highest score wins):

| Component | Condition | Score |
|-----------|-----------|-------|
| `product-grid` | 2+ `column_groups`, each has image + text | 0.95 |
| `article-card` | 1 image + ≥2 texts, single column (no multi-column groups) | 0.9 |
| `image-grid` | 2 images, ≤1 text | 0.85 |
| `image-gallery` | 3+ images, ≤1 text | 0.88 |
| `category-nav` | 3+ texts all <20 chars, ≤1 image | 0.7 |
| `navigation-bar` | `_all_images_are_icons(section)` and has_texts | 0.9 |
| `image-block` | 1 image only | 1.0 |
| `text-block` | texts only (with or without buttons/headings) | 1.0 |
| `spacer` | no content | 0.5 |

Implementation approach:
```python
def _score_candidates(section, has_images, has_texts, has_buttons, has_headings):
    candidates: list[tuple[str, float]] = []

    img_count = len(section.images)
    text_count = len(section.texts)
    col_groups = section.column_groups or []
    groups_with_mixed = sum(
        1 for g in col_groups if g.images and g.texts
    )

    # product-grid: 2+ column groups each with image + text
    if len(col_groups) >= 2 and groups_with_mixed >= 2:
        candidates.append(("product-grid", 0.95))

    # image-gallery: 3+ images, minimal text
    if img_count >= 3 and text_count <= 1:
        candidates.append(("image-gallery", 0.88))

    # image-grid: exactly 2 images, minimal text
    if img_count >= 2 and text_count <= 1 and img_count < 3:
        candidates.append(("image-grid", 0.85))

    # article-card: 1 image + 2+ texts, no multi-column groups
    if img_count == 1 and text_count >= 2 and len(col_groups) <= 1:
        candidates.append(("article-card", 0.9))

    # navigation-bar: icons + text
    if has_images and has_texts and _all_images_are_icons(section):
        candidates.append(("navigation-bar", 0.9))

    # category-nav: 3+ short texts
    short_texts = [t for t in section.texts if len(t.content) < 20]
    if len(short_texts) >= 3 and img_count <= 1:
        candidates.append(("category-nav", 0.7))

    # image-block: single image, no text
    if img_count == 1 and not has_texts:
        candidates.append(("image-block", 1.0))

    # text-block: text only
    if has_texts and not has_images:
        candidates.append(("text-block", 1.0))

    if not candidates:
        return ("text-block", 0.5)  # fallback

    # Sort by score descending, return best
    candidates.sort(key=lambda c: c[1], reverse=True)
    best_slug, best_score = candidates[0]

    if best_score < 0.5:
        return ("text-block", 0.5)

    return (best_slug, best_score)
```

Update all call sites of `_match_content()` to use `_score_candidates()`.

### Step 3: Spatial Column Assignment

**File:** `component_matcher.py:602–641`

Replace round-robin fallback in `_build_column_fills()` with spatial bucketing:

```python
def _assign_spatial_column(
    x_position: float | None,
    col_count: int,
    section_width: float | None,
) -> int:
    """Return 1-indexed column for an item based on X position."""
    if x_position is None:
        return 1
    width = section_width or 600.0
    col_width = width / col_count
    col_idx = int(x_position / col_width) + 1
    return max(1, min(col_idx, col_count))  # clamp
```

In `_build_column_fills()`, when no `column_groups`:
- For each text/image/button, use `_assign_spatial_column(item.x_position, ...)` instead of `i % col_count`
- Group items by assigned column
- Build fills per column in spatial order

Note: Check if `TextBlock`/`ImagePlaceholder`/`ButtonElement` have `x_position` attribute. If not, add it to `layout_analyzer.py` dataclasses (they likely have it from Phase 39.1 Figma API enrichment).

```python
def _fills_product_grid(section, container_width, *, image_urls=None, **_kw):
    fills: list[SlotFill] = []
    groups = section.column_groups or []
    for i, group in enumerate(groups[:4], 1):  # max 4 products
        if group.images:
            img = group.images[0]
            fills.append(SlotFill(f"product_{i}_image", _resolve_image_url(img.node_id, image_urls), "image"))
            fills.append(SlotFill(f"product_{i}_image_alt", img.node_name, "text"))
        heading = _first_heading(group.texts)
        if heading:
            fills.append(SlotFill(f"product_{i}_title", _safe_text(heading.content), "text"))
        body = _body_texts(group.texts)
        if body:
            fills.append(SlotFill(f"product_{i}_desc", _safe_text(body[0].content), "text"))
        if group.buttons:
            fills.append(SlotFill(f"product_{i}_cta", _safe_text(group.buttons[0].text), "cta"))
    return fills

def _fills_category_nav(section, _cw, **_kw):
    fills: list[SlotFill] = []
    for i, text in enumerate(section.texts[:6], 1):
        fills.append(SlotFill(f"nav_item_{i}", _safe_text(text.content), "text"))
    return fills

def _fills_image_gallery(section, _cw, *, image_urls=None, **_kw):
    fills: list[SlotFill] = []
    for i, img in enumerate(section.images[:6], 1):
        fills.append(SlotFill(f"image_{i}", _resolve_image_url(img.node_id, image_urls), "image"))
        fills.append(SlotFill(f"image_{i}_alt", img.node_name, "text"))
    return fills
```

### Step 6: Post-Render Slot Validation

**File:** `component_renderer.py`

Add validation function:
```python
import re

def _validate_slot_fill_rate(
    template_html: str,
    slot_fills: list[SlotFill],
) -> tuple[float, list[str]]:
    slot_ids = set(re.findall(r'data-slot="([^"]+)"', template_html))
    total = len(slot_ids)
    if total == 0:
        return 1.0, []
    filled = sum(1 for f in slot_fills if f.slot_id in slot_ids)
    rate = filled / total
    warnings: list[str] = []
    if rate < 0.5:
        warnings.append(f"Low slot fill rate ({filled}/{total} = {rate:.0%})")
    return rate, warnings
```

Call in `render_section()` after `_fill_slots()`:
