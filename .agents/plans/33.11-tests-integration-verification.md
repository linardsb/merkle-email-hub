1. **E2E MJML pipeline test** — No test covers full `normalize → analyze → classify → MJML generate → compile → verify HTML`

4. **Sidecar MJML edge cases** — No dedicated MJML compile test file (large input, error array format)
5. **Visual fidelity edge cases** — Missing: single-pixel diff tolerance, width mismatch resize, empty sections list

### Step 1: E2E MJML Pipeline Test
**File:** `app/design_sync/tests/test_e2e_mjml_pipeline.py` (~300 lines, 15 tests)

Build a realistic 4-section structure (Header + Hero + 2-col Content + Footer) with factories following `test_e2e_pipeline.py` patterns. Mock `compile_mjml()` HTTP call + LLM calls.

| Test | Verifies |
|------|----------|
| `test_full_pipeline_normalize_to_mjml` | `normalize_tree` → `analyze_layout` → `generate_mjml` → valid `<mjml>` document |
| `test_pipeline_template_engine_render` | `MjmlTemplateEngine.render_email()` → valid MJML with all sections |
| `test_pipeline_section_count_matches` | Layout section count == MJML `<mj-section>` count |
| `test_pipeline_dark_mode_css` | Dark tokens → `<mj-style>` with `@media (prefers-color-scheme: dark)` |
| `test_pipeline_token_injection` | Palette colors in `mj-section background-color`, typography in `mj-text font-family` |
| `test_pipeline_multi_column` | 2-col section → `<mj-column width="50%">` x2 |
| `test_pipeline_section_markers` | `<!-- section:frame_id:HEADER -->` comments in MJML |
| `test_pipeline_preheader` | Preheader text in `<mj-preview>` |
| `test_pipeline_responsive_attrs` | `<mj-body width="600">` present |
| `test_pipeline_empty_structure` | Empty tree → minimal MJML (head only) |
| `test_pipeline_single_cta_section` | One CTA → valid single-section MJML |
| `test_pipeline_normalize_removes_hidden` | Hidden nodes excluded from layout |
| `test_pipeline_classify_unknown_sections` | UNKNOWN → AI classifier (mocked) reclassifies |
| `test_pipeline_convert_mjml_service` | `DesignConverterService.convert_mjml()` e2e with mocked sidecar |
| `test_pipeline_fallback_on_compile_error` | `MjmlCompileError` → fallback to `_convert_recursive()` HTML |

**Factory pattern:**
```python
def _make_node(name: str, ntype: DesignNodeType, *, children: list | None = None, **kw) -> DesignNode: ...
def _make_tokens(*, colors: list | None = None, dark_colors: list | None = None) -> ExtractedTokens: ...
def _make_structure(pages: list | None = None) -> DesignFileStructure: ...
```

**Mock strategy:** `@patch("app.design_sync.converter_service.httpx.AsyncClient")` for MJML compile. `@patch("app.design_sync.ai_layout_classifier.get_ai_provider")` for classifier.
