# Plan: 36.1 EmailDesignDocument JSON Schema v1

## Context

The pipeline passes Python objects in-memory between providers and the converter — tight coupling, no validation, hard to test without a provider. A formal JSON Schema makes the contract explicit, enables validation at boundaries, allows JSON fixtures for testing, supports caching/versioning/diffing, and opens the door for external tools targeting the schema via API.

### Current Data Flow

```
Provider (Figma/Penpot) → ExtractedTokens + DesignFileStructure
  → asdict(tokens) → DesignTokenSnapshot.tokens_json (raw JSON, no validation)
  → DesignConverterService.convert() consumes ExtractedTokens + DesignLayoutDescription separately
  → mjml_generator.py / mjml_template_engine.py / converter.py produce HTML
```

### Step 2: Create JSON Schema file

`data/schemas/email-design-document-v1.json` — Draft 2020-12 with:

- **Root:** `version` (const "1.0"), `source?`, `tokens`, `sections[]`, `layout`, `compatibility_hints[]?`, `token_warnings[]?`
- **`$defs/source`:** `provider` (enum: figma/penpot/mjml/html/manual/sketch/canva), `file_ref`, `synced_at` (date-time)
- **`$defs/tokens`:** All optional arrays — `colors[]` (name, hex pattern `^#[0-9A-Fa-f]{6}$`, opacity 0-1, role?), `typography[]` (name, family, size ≥0, weight, line_height, letter_spacing, text_transform?, text_decoration?), `spacing[]` (name, value ≥0), `dark_colors[]`, `gradients[]` (angle 0-360, stops[] with hex+position, fallback_hex), `variables[]` (name, type, value, mode?), `stroke_colors[]`
- **`$defs/section`:** Required: id, type (enum 11 values). Optional: column_layout (enum 4), width, height, padding (top/right/bottom/left each ≥0), item_spacing, background_color, `texts[]` (maxItems 50), `images[]`, `buttons[]`, `columns[]` (width_pct, texts, images, buttons), content_roles[], spacing_after, classification_confidence 0-1, node_name, y_position
- **`$defs/text`:** content (required), is_heading, font_family, font_size ≥0, font_weight, color, line_height, letter_spacing, text_transform, text_decoration, node_id
- **`$defs/image`:** node_id (required), width, height, alt, src, node_name, is_background
- **`$defs/button`:** text (required), url, background_color, text_color, border_radius ≥0, width, height, node_id, padding
- **`$defs/layout`:** container_width (int 400-800, default 600), naming_convention?, overall_width?
- **`$defs/compatibility_hint`:** level (enum: info/warning/error), css_property, message, affected_clients[]
- **`$defs/token_warning`:** level (enum: error/warning/info), field, message, fixed_value?
- **Limits:** sections maxItems 100, colors maxItems 500, typography maxItems 200, texts per section maxItems 50, images per section maxItems 50, buttons per section maxItems 30
- **`additionalProperties: false`** on all inner objects

@dataclass(frozen=True)
class DocumentSpacing:

class DocumentGradientStop:
    hex: str
    position: float  # 0-1

@dataclass(frozen=True)
class DocumentGradient:
    angle: float
    stops: list[DocumentGradientStop]

@dataclass(frozen=True)

class DocumentButton:
    text: str
    url: str | None = None
    background_color: str | None = None
    text_color: str | None = None
    border_radius: float | None = None
    width: float | None = None
    height: float | None = None

class DocumentColumn:

    texts: list[DocumentText] = field(default_factory=list)

    buttons: list[DocumentButton] = field(default_factory=list)

@dataclass(frozen=True)
class DocumentSectionPadding:
    top: float = 0.0
    right: float = 0.0
    bottom: float = 0.0
    left: float = 0.0

@dataclass(frozen=True)
class DocumentSection:
    id: str
    type: str  # EmailSectionType enum value
    column_layout: str = "single"
    width: float | None = None
    height: float | None = None
    padding: DocumentSectionPadding | None = None
    item_spacing: float | None = None
    background_color: str | None = None
    texts: list[DocumentText] = field(default_factory=list)
    images: list[DocumentImage] = field(default_factory=list)
    buttons: list[DocumentButton] = field(default_factory=list)
    columns: list[DocumentColumn] = field(default_factory=list)

    classification_confidence: float | None = None
    node_name: str | None = None
    y_position: float | None = None

    def to_email_section(self) -> EmailSection: ...  # Bridge method

@dataclass(frozen=True)
class DocumentLayout:
    container_width: int = 600
    naming_convention: str | None = None
    overall_width: float | None = None

@dataclass(frozen=True)
class CompatibilityHint:
    level: str  # info, warning, error
    css_property: str
    message: str
    affected_clients: list[str] = field(default_factory=list)

@dataclass(frozen=True)
class TokenWarning:
    level: str  # error, warning, info
    field: str
    message: str
    fixed_value: str | None = None

@dataclass(frozen=True)
class EmailDesignDocument:
    version: str
    tokens: DocumentTokens
    sections: list[DocumentSection]
    layout: DocumentLayout
    source: DocumentSource | None = None
    compatibility_hints: list[CompatibilityHint] = field(default_factory=list)
    token_warnings: list[TokenWarning] = field(default_factory=list)

    @classmethod
    def from_json(cls, data: dict[str, Any]) -> EmailDesignDocument: ...
    def to_json(self) -> dict[str, Any]: ...

    @staticmethod
    def validate(data: dict[str, Any]) -> list[str]: ...
    # Uses jsonschema.validate() against the JSON Schema file
    # Returns list of error messages (empty = valid)

    @staticmethod
    def schema() -> dict[str, Any]: ...
    # Loads and returns the JSON Schema (cached via @lru_cache)

    def to_extracted_tokens(self) -> ExtractedTokens: ...
    def to_email_sections(self) -> list[EmailSection]: ...
    def to_layout_description(self, file_name: str = "") -> DesignLayoutDescription: ...

## Security Checklist

| Endpoint | Auth | Rate Limit | Validation | Notes |
|----------|------|------------|------------|-------|
| `POST /validate-document` | `get_current_user` | 30/min | JSON Schema + 5MB max body | Returns validation errors only, no internal types |
| `GET /schema/v1` | None (public) | 60/min | N/A | Static file, Cache-Control |

- `additionalProperties: false` rejects unknown fields — prevents data smuggling
- `maxItems` on all arrays — prevents DoS via oversized documents
- Max body size 5MB — enforced before JSON parsing
- Hex color pattern validation `^#[0-9A-Fa-f]{6}$` — prevents injection via color fields
- `from_json()` raises `ValueError` on malformed data — no bare `Exception`
- No `eval()`, no `exec()`, no dynamic attribute access from user input
