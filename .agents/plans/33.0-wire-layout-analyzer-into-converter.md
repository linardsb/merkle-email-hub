# Plan: 33.0 Wire Layout Analyzer into Converter

## Context

`DesignConverterService.convert()` calls `node_to_email_html(frame, indent=1, props_map=...)` on raw `DesignNode` trees (line 110), bypassing the layout analyzer entirely. The analyzer (`analyze_layout()` in `app/design_sync/figma/layout_analyzer.py`) already detects column layouts, buttons, heading hierarchy, item spacing, element gaps, and section types — but none of this data reaches HTML generation. Every Phase 33 subtask (33.3–33.9) needs this data.

Additionally:
- `_build_props_map_from_nodes()` only extracts `bg_color`, silently discarding `font_family`, `font_size`, `font_weight`, `padding_*`, `layout_mode`, `line_height_px`, `letter_spacing_px`
- `EMAIL_SKELETON` is missing required meta tags and MSO reset styles
- Inline `<table>` and `<img>` styles lack MSO-specific properties needed for Gmail clipping resilience

## Files to Create/Modify

- `app/design_sync/converter.py` — extend `_NodeProps`, update `node_to_email_html()` signature + table/image output + nesting depth guard
- `app/design_sync/converter_service.py` — wire `analyze_layout()` into `convert()`, fix `_build_props_map_from_nodes()`, update `EMAIL_SKELETON`, extend `ConversionResult`
- `app/design_sync/tests/test_penpot_converter.py` — update skeleton tests (new format args), add new tests for MSO styles, nesting depth, layout integration
- `app/design_sync/tests/test_layout_analyzer.py` — no changes needed (layout analyzer tests are independent)

```python
def _build_props_map_from_nodes(self, frames: list[DesignNode]) -> dict[str, _NodeProps]:
    """Build props_map from DesignNode fields (provider-agnostic)."""
    props: dict[str, _NodeProps] = {}

    def _walk(node: DesignNode) -> None:
        has_data = bool(
            node.fill_color or node.font_family or node.font_size
            or node.font_weight or node.padding_top or node.padding_right
            or node.padding_bottom or node.padding_left or node.layout_mode
            or node.line_height_px or node.letter_spacing_px
        )
        if has_data:
            props[node.id] = _NodeProps(
                bg_color=node.fill_color,
                font_family=node.font_family,
                font_size=node.font_size,
                font_weight=str(node.font_weight) if node.font_weight else None,
                padding_top=node.padding_top or 0,
                padding_right=node.padding_right or 0,
                padding_bottom=node.padding_bottom or 0,
                padding_left=node.padding_left or 0,
                layout_direction=(
                    "row" if node.layout_mode == "HORIZONTAL"
                    else "column" if node.layout_mode == "VERTICAL"
                    else None
                ),
                line_height_px=node.line_height_px,
                letter_spacing_px=node.letter_spacing_px,
            )
        for child in node.children:
            _walk(child)

    for frame in frames:
        _walk(frame)
    return props
```
