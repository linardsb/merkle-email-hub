The hub tests 7 golden templates — a tiny fraction of the possible configuration space. Property-based testing (QuickCheck/Hypothesis-style) generates hundreds of random email configurations and asserts that invariants hold across all of them. This catches edge cases that no human would think to test.

### Step 4: Create `invariants.py`

Define the `EmailInvariant` Protocol and 10 pre-built invariants.

```python
"""Email invariants — properties that must hold regardless of content."""

from __future__ import annotations

import re
from dataclasses import dataclass
from typing import Protocol

from lxml import html as lxml_html
from lxml.html import HtmlElement


@dataclass(frozen=True)
class InvariantResult:
    """Result of checking a single invariant."""

    invariant_name: str
    passed: bool
    violations: tuple[str, ...] = ()


class EmailInvariant(Protocol):
    """Protocol for email invariants."""

    name: str

    def check(self, html: str) -> InvariantResult: ...
```

**10 invariants to implement:**

1. **`SizeLimit`** — `len(html.encode("utf-8")) < 102_400` (Gmail 102KB clipping). Simple byte length check.

2. **`ImageWidth`** — Parse with lxml, find all `<img>` tags, check `width` attribute (if present) is ≤ 600. Also check inline `style` for `width:` > 600px. Violations list each offending image (by src or position).

3. **`LinkIntegrity`** — Every `<a>` has non-empty `href`. No `javascript:` URIs. Use lxml `doc.iter("a")`.

4. **`AltTextPresence`** — Every `<img>` has non-empty `alt` attribute. Skip tracking pixels (1x1 images). Use lxml `doc.iter("img")`.

5. **`TableNestingDepth`** — Walk table elements, compute max nesting depth. Violation if > 8 (Outlook rendering limit). Recursive traversal with `element.getparent()` or walk-down counting.

6. **`EncodingValid`** — All characters valid UTF-8, no null bytes. `"\x00" not in html`. Try `html.encode("utf-8")` — catch `UnicodeEncodeError`.

7. **`MSOBalance`** — Every `<!--[if mso]>` has a matching `<![endif]-->`. Count with regex: `r"<!--\[if\s+[^\]]*\]>"` vs `r"<!\[endif\]-->"`. Also check `<!--[if !mso]><!-->`/`<!--<![endif]-->` pairs.

8. **`DarkModeReady`** — If `prefers-color-scheme` appears in CSS, both light and dark values must be specified. Parse `<style>` blocks, look for `@media (prefers-color-scheme: dark)`. If present, verify there's a corresponding light color scheme or that color-scheme meta is present.

9. **`ContrastRatio`** — Check inline `color` + `background-color` pairs on elements. Compute relative luminance ratio per WCAG formula. Violation if ratio < 4.5:1. Use regex to extract hex colors from inline styles. This is best-effort (can't resolve all CSS cascading) — only checks explicit inline pairs.

10. **`ViewportFit`** — Check that no element has explicit `width` > 600px in inline styles. Also check `<table>` width attributes. Similar to ImageWidth but for all elements.

**Implementation pattern** (same for all invariants):

```python
class SizeLimit:
    name = "size_limit"

    def check(self, html: str) -> InvariantResult:
        size = len(html.encode("utf-8"))
        if size > 102_400:
            return InvariantResult(
                invariant_name=self.name,
                passed=False,
                violations=(f"HTML size {size} bytes exceeds 102,400 byte Gmail clipping threshold",),
            )
        return InvariantResult(invariant_name=self.name, passed=True)
```

Export a registry:

```python

    inv.name: inv
    for inv in [
        SizeLimit(), ImageWidth(), LinkIntegrity(), AltTextPresence(),
        TableNestingDepth(), EncodingValid(), MSOBalance(),
        DarkModeReady(), ContrastRatio(), ViewportFit(),

}
```

### Step 5: Create `generators.py`

Use Hypothesis strategies to generate random email HTML. The key insight: we don't need `TemplateAssembler` (it requires golden templates and full EmailBuildPlan). Instead, generate synthetic email HTML directly using strategies that produce realistic email structures.

```python
"""Hypothesis strategies for generating random email HTML."""

from __future__ import annotations

from dataclasses import dataclass

from hypothesis import strategies as st
from hypothesis.strategies import SearchStrategy


@dataclass(frozen=True)
class EmailConfig:
    """Configuration for a generated email."""

    section_count: int
    section_types: tuple[str, ...]
    content_lengths: tuple[int, ...]
    font_family: str
    primary_color: str
    background_color: str
    text_color: str
    image_count: int
    image_widths: tuple[int, ...]
    table_nesting_depth: int
    has_mso_conditionals: bool
    has_dark_mode: bool
    dark_mode_complete: bool  # Whether dark mode has both light+dark values
    link_count: int
    include_javascript_links: bool  # For testing LinkIntegrity
    include_empty_alt: bool  # For testing AltTextPresence
    target_size_kb: int  # Approximate target size
```

**Strategies:**

```python
SECTION_TYPES = ("hero", "content", "cta", "footer", "navigation", "divider", "image_row", "social")
FONT_FAMILIES = ("Arial, sans-serif", "Georgia, serif", "Helvetica, Arial, sans-serif", "'Trebuchet MS', sans-serif")

def email_configs() -> SearchStrategy[EmailConfig]:
    """Strategy producing random email configurations."""
    return st.builds(
        EmailConfig,
        section_count=st.integers(min_value=1, max_value=15),
        section_types=st.lists(st.sampled_from(SECTION_TYPES), min_size=1, max_size=15).map(tuple),
        content_lengths=st.lists(st.integers(min_value=10, max_value=2000), min_size=1, max_size=15).map(tuple),
        font_family=st.sampled_from(FONT_FAMILIES),
        primary_color=_hex_colors(),
        background_color=_hex_colors(),
        text_color=_hex_colors(),
        image_count=st.integers(min_value=0, max_value=10),
        image_widths=st.lists(st.integers(min_value=50, max_value=800), min_size=0, max_size=10).map(tuple),
        table_nesting_depth=st.integers(min_value=0, max_value=12),
        has_mso_conditionals=st.booleans(),
        has_dark_mode=st.booleans(),
        dark_mode_complete=st.booleans(),
        link_count=st.integers(min_value=0, max_value=20),
        include_javascript_links=st.booleans(),
        include_empty_alt=st.booleans(),
        target_size_kb=st.integers(min_value=5, max_value=150),
    )


def _hex_colors() -> SearchStrategy[str]:
    """Strategy producing valid hex color strings."""
    return st.from_regex(r"#[0-9a-fA-F]{6}", fullmatch=True)
```

**`build_email(config: EmailConfig) -> str`** — Pure function that builds HTML from config:

```python
def build_email(config: EmailConfig) -> str:
    """Build synthetic email HTML from configuration."""
    parts: list[str] = []
    parts.append('<!DOCTYPE html>')
    parts.append('<html lang="en" xmlns:v="urn:schemas-microsoft-com:vml">')
    parts.append("<head>")
    parts.append('<meta charset="utf-8">')
    parts.append('<meta name="viewport" content="width=device-width, initial-scale=1.0">')
    parts.append("<title>Test Email</title>")

    # Dark mode support
    if config.has_dark_mode:
        parts.append('<meta name="color-scheme" content="light dark">')
        parts.append("<style>")
        if config.dark_mode_complete:
            parts.append("@media (prefers-color-scheme: dark) {")
            parts.append(f"  body {{ background-color: #1a1a2e; color: #ffffff; }}")
            parts.append("}")
        else:
            # Incomplete — only dark, no light counterpart in @media
            parts.append("@media (prefers-color-scheme: dark) {")
            parts.append(f"  body {{ background-color: #1a1a2e; }}")
            parts.append("}")
        parts.append("</style>")

    parts.append("</head>")
    parts.append(f'<body style="background-color:{config.background_color}; color:{config.text_color}; font-family:{config.font_family};">')

    # MSO conditionals
    if config.has_mso_conditionals:
        parts.append("<!--[if mso]>")
        parts.append('<table role="presentation" width="600"><tr><td>')
        parts.append("<![endif]-->")

    # Sections with nested tables
    parts.append(_build_sections(config))

    # Images
    for i in range(config.image_count):
        width = config.image_widths[i] if i < len(config.image_widths) else 300
        alt = "" if config.include_empty_alt and i == 0 else f"Image {i+1}"
        parts.append(f'<img src="https://example.com/img{i}.png" width="{width}" alt="{alt}" style="width:{width}px;">')

    # Links
    for i in range(config.link_count):
        if config.include_javascript_links and i == 0:
            parts.append('<a href="javascript:void(0)">Click</a>')
        else:
            parts.append(f'<a href="https://example.com/link{i}">Link {i}</a>')

    # Close MSO
    if config.has_mso_conditionals:
        parts.append("<!--[if mso]>")
        parts.append("</td></tr></table>")
        parts.append("<![endif]-->")

    parts.append("</body></html>")

    html = "\n".join(parts)

    # Pad to target size if needed
    if config.target_size_kb > 0:
        target_bytes = config.target_size_kb * 1024
        current = len(html.encode("utf-8"))
        if current < target_bytes:
            padding = "<!-- padding " + "x" * (target_bytes - current - 20) + " -->"
            html = html.replace("</body>", padding + "\n</body>")

    return html
```

**`_build_sections(config)`** — builds nested tables to the specified depth:

```python
def _build_sections(config: EmailConfig) -> str:
    """Build section HTML with table nesting."""
    parts: list[str] = []
    depth = min(config.table_nesting_depth, config.section_count)
    for i in range(depth):
        content_len = config.content_lengths[i] if i < len(config.content_lengths) else 100
        content = "L" * content_len  # Lorem placeholder
        parts.append(f'<table role="presentation" width="600"><tr><td style="color:{config.text_color}; background-color:{config.background_color};">')
        parts.append(f"<p>{content}</p>")
    # Close tables in reverse
    for _ in range(depth):
        parts.append("</td></tr></table>")
    # Remaining sections (flat)
    for i in range(depth, config.section_count):
        content_len = config.content_lengths[i] if i < len(config.content_lengths) else 100
        content = "L" * content_len
        parts.append(f'<table role="presentation" width="600"><tr><td>')
        parts.append(f"<p>{content}</p>")
        parts.append("</td></tr></table>")
    return "\n".join(parts)
```

### Step 6: Create `runner.py`

```python
"""Property test runner — generates emails, checks invariants, reports failures."""

from __future__ import annotations

import random
from dataclasses import dataclass, field

from app.core.logging import get_logger
from app.qa_engine.property_testing.generators import EmailConfig, build_email, email_configs
from app.qa_engine.property_testing.invariants import ALL_INVARIANTS, InvariantResult

logger = get_logger(__name__)


@dataclass(frozen=True)
class PropertyFailure:
    """A single invariant failure with the config that caused it."""

    invariant_name: str
    violations: tuple[str, ...]
    config: EmailConfig  # The minimal config that triggers the violation


@dataclass(frozen=True)
class PropertyTestReport:
    """Results of a property test run."""

    total_cases: int
    passed: int
    failed: int
    failures: tuple[PropertyFailure, ...] = ()
    seed: int = 0


class PropertyTestRunner:
    """Generates random emails and checks invariants."""

    async def run(
        self,
        invariant_names: list[str] | None = None,
        num_cases: int = 100,
        seed: int | None = None,
    ) -> PropertyTestReport:
        """Run property tests.

        Args:
            invariant_names: Invariants to check. None = all.
            num_cases: Number of random emails to generate.
            seed: Fixed seed for reproducibility. None = random.

        Returns:
            PropertyTestReport with failures and minimal configs.
        """
        actual_seed = seed if seed is not None else random.randint(0, 2**32 - 1)

        # Resolve invariants
        if invariant_names:
            invariants = [ALL_INVARIANTS[n] for n in invariant_names if n in ALL_INVARIANTS]
        else:
            invariants = list(ALL_INVARIANTS.values())

        if not invariants:
            return PropertyTestReport(total_cases=0, passed=0, failed=0, seed=actual_seed)

        # Generate configs using hypothesis with fixed seed
        strategy = email_configs()
        configs: list[EmailConfig] = []
        rng = random.Random(actual_seed)
        from hypothesis import settings as h_settings, given, HealthCheck
        from hypothesis.database import InMemoryExampleDatabase
        from hypothesis.strategies import DataObject

        # Use hypothesis find() to generate examples deterministically
        from hypothesis import find as h_find

        for i in range(num_cases):
            # Use a sub-seed per case for determinism
            sub_seed = rng.randint(0, 2**32 - 1)
            try:
                config = strategy.example()  # Will be replaced with seeded generation below
            except Exception:
                continue
            configs.append(config)

        # Check invariants
        failures: list[PropertyFailure] = []
        passed_count = 0
        failed_count = 0

        for config in configs:
            html = build_email(config)
            case_failed = False

            for invariant in invariants:
                try:
                    result = invariant.check(html)
                except Exception as exc:
                    logger.warning(
                        "property_test.invariant_error",
                        invariant=invariant.name,
                        error=str(exc),
                    )
                    continue

                if not result.passed:
                    case_failed = True
                    failures.append(
                        PropertyFailure(
                            invariant_name=invariant.name,
                            violations=result.violations,
                            config=config,
                        )
                    )

            if case_failed:
                failed_count += 1
            else:
                passed_count += 1

        logger.info(
            "property_test.completed",
            total=len(configs),
            passed=passed_count,
            failed=failed_count,
            unique_invariant_failures=len({f.invariant_name for f in failures}),
            seed=actual_seed,
        )

        return PropertyTestReport(
            total_cases=len(configs),
            passed=passed_count,
            failed=failed_count,
            failures=tuple(failures),
            seed=actual_seed,
        )
```

**Important design decision:** Don't use `strategy.example()` (non-deterministic). Instead, use Hypothesis's `Random` parameter seeding:

```python
from hypothesis.strategies import SearchStrategy
from hypothesis._settings import default_settings
from hypothesis.internal.conjecture.data import ConjectureData
from hypothesis.internal.entropy import deterministic_PRNG

# Better approach: use the strategy's draw with a fixed random
import struct

def _draw_examples(strategy: SearchStrategy[EmailConfig], count: int, seed: int) -> list[EmailConfig]:
    """Draw deterministic examples from a Hypothesis strategy."""
    results: list[EmailConfig] = []
    rng = random.Random(seed)
    for _ in range(count):
        sub_seed = rng.getrandbits(64)
        try:
            # Hypothesis provides a way to generate from a seed via find()
            # For deterministic generation, use the Random object
            example = strategy.example()
            results.append(example)
        except Exception:
            continue
    return results
```

**Actual implementation note:** For true deterministic seeded generation, the runner should use `random.seed(actual_seed)` before calling `strategy.example()` in a loop. Hypothesis's `example()` uses the global random state. This is sufficient for reproducibility. The full implementation should:

1. `random.seed(actual_seed)` at the start
2. Loop `num_cases` times calling `strategy.example()`
3. Build email from each config
4. Check all invariants
5. Collect failures
6. Restore random state after

```python
async def run_property_test(self, data: PropertyTestRequest) -> PropertyTestResponse:
    """Run property-based testing on randomly generated emails."""
    settings = get_settings()
    if not settings.qa_property_testing.enabled:
        raise ForbiddenError("Property testing is not enabled")

    from app.qa_engine.property_testing.runner import PropertyTestRunner

    runner = PropertyTestRunner()
    seed = data.seed if data.seed is not None else settings.qa_property_testing.seed
    num_cases = data.num_cases or settings.qa_property_testing.default_cases
    result = await runner.run(
        invariant_names=data.invariants,
        num_cases=num_cases,
        seed=seed,
    )

    logger.info(
        "qa_engine.property_test_completed",
        total_cases=result.total_cases,
        passed=result.passed,
        failed=result.failed,
        seed=result.seed,
    )

    # Convert to response
    from dataclasses import asdict
    failures = [
        PropertyFailureSchema(
            invariant_name=f.invariant_name,
            violations=list(f.violations),
            config=asdict(f.config),
        )
        for f in result.failures
    ]

    return PropertyTestResponse(
        total_cases=result.total_cases,
        passed=result.passed,
        failed=result.failed,
        failures=failures,
        seed=result.seed,
        invariants_tested=data.invariants or list(ALL_INVARIANTS.keys()),
    )
```
