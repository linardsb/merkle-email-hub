# Plan: 1.5 Test Persona Engine UI

## Context
The workspace preview panel has manual viewport (desktop/tablet/mobile) and dark mode toggles, but no way to simulate specific email client environments. Task 1.5 adds a persona selector to the preview toolbar so developers can one-click preview as "Gmail Desktop", "iPhone Mail", "Outlook 365", etc. The backend Persona API is complete (models, routes, service) and 8 presets are defined — but not yet seeded into the database.

## Architecture Decision
Selecting a persona applies its `viewport_width`, `dark_mode`, and `device_type` as the preview context. The existing manual viewport/dark mode toggles remain functional — they show the persona's values when one is selected, and the user can still toggle them independently (deselecting the persona). This keeps the persona as a "quick preset" without adding complex linking logic.

### Step 6: Update PreviewPanel to manage persona state

**File:** `cms/apps/web/src/components/workspace/preview-panel.tsx`

Add persona props:
```typescript
interface PreviewPanelProps {
  compiledHtml: string | null;
  isCompiling: boolean;
  buildTimeMs: number | null;
  onCompile: () => void;
  hasContent: boolean;
  personas: PersonaResponse[];
  selectedPersonaId: number | null;
  onPersonaSelect: (persona: PersonaResponse | null) => void;
  isLoadingPersonas?: boolean;
}
```

When a persona is selected, derive viewport and dark mode from the persona:
- Find `selectedPersona` from `personas` array by `selectedPersonaId`
- Map persona `viewport_width` to closest Viewport enum for toolbar highlight:
  - `<= 400` → "mobile", `<= 768` → "tablet", else → "desktop"
- Map persona `dark_mode` boolean to dark mode state
- When user manually changes viewport or dark mode → deselect persona (`onPersonaSelect(null)`)
- Pass `viewportWidthOverride={selectedPersona?.viewport_width ?? null}` to PreviewIframe

Handler for persona selection:
```typescript
const handlePersonaSelect = useCallback((persona: PersonaResponse | null) => {
  onPersonaSelect(persona);
  if (persona) {
    setViewport(mapWidthToViewport(persona.viewport_width ?? 600));
    setDarkMode(persona.dark_mode ?? false);
  }
}, [onPersonaSelect]);
```

Handler for manual viewport/dark mode changes — deselect persona:
```typescript
const handleViewportChange = useCallback((v: Viewport) => {
  setViewport(v);
  onPersonaSelect(null); // clear persona on manual change
}, [onPersonaSelect]);

const handleDarkModeToggle = useCallback(() => {
  setDarkMode(d => !d);
  onPersonaSelect(null); // clear persona on manual change
}, [onPersonaSelect]);
```
