# Plan: 52.9 Rule 5 — Asset Name vs Content Mismatch (needs PNG) — STUB

**Status:** Deferred. Detailed plan promotes when 52.9 starts.

## Summary

When emitting `data-slot` candidates, base assignment on (1) **aspect ratio** (very wide → banner; ≤ 1:1.2 → square icon; ≥ 1:4 → barcode/strip), (2) **sibling order within parent card** (top → header logo, middle → content, bottom → decoration / shape), (3) **image-fill perceptual hash** if a brand asset library is available. `name` field used only as tie-breaker.

Canonical case: LEGO `footer-card.png` (actually the Insiders header logo with shapes), `footer-social.png` (actually the barcode), `footer-logo.png` (actually the bottom blue paper-cut shape).
