# Plan: 51.3 Tag/Pill Slot + Role Detection (Gap 2) — STUB

**Status:** Deferred. Detailed plan promotes when 51.3 starts.

## Summary

`_extract_content_groups` marks any text whose parent FRAME is a small fill+padding leaf (`layoutMode=HORIZONTAL`, `paddingLeft/Right ≥ 8`, single TEXT child) as `role_hint="tag"`. **Detection does NOT gate on `cornerRadius`** (per Rule 8 in 50.5 — pills can be square). Tag slot template emits alignment from Rule 7 + radius from Rule 8 (both in 50.5).

Adds optional `tag` slot to `article-card.html`, `zigzag-image-{left,right}.html`, editorial cards. When unfilled, the tag row is stripped (already standard component hygiene).
