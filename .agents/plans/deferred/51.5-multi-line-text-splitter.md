# Plan: 51.5 Multi-Line Text Splitter (Gap 12) — STUB

**Status:** Deferred. Detailed plan promotes when 51.5 starts.

## Summary

`_extract_texts` detects `\n` in TEXT node content and splits into structurally-paired children: first half `role_hint="value"` + second half `role_hint="label"`, plus `font_weight_diff` flag. `_safe_text` replaces `\n` with `<br>` after HTML-escape (fallback for un-split nodes).

Canonical case: spec mini-table value `'617\nPieces'` — single TEXT node with literal newline, Figma renders as two visually distinct lines (number on top, label below).
