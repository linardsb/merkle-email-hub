## Schemas

```python
class ExportRequest(BaseModel):
    html: str | None = Field(None, max_length=5_000_000)
    template_id: uuid.UUID | None = None  # fetch HTML from DB if set
    target_esp: str
    connection_id: uuid.UUID
    template_name: str = "Exported Email"
    source_esp: str | None = None
    rewrite_tokens: bool = True

    @model_validator(mode="after")
    def require_html_or_template_id(self) -> Self:
        if not self.html and not self.template_id:
            raise ValueError("Either html or template_id required")
        return self

class ExportResponse(BaseModel):
    esp_template_id: str
    template_name: str
    target_esp: str
    tokens_rewritten: int
    warnings: list[str]

class BulkExportRequest(BaseModel):
    template_ids: list[uuid.UUID] = Field(..., min_length=1, max_length=50)
    target_esp: str
    connection_id: uuid.UUID
    rewrite_tokens: bool = True

class BulkExportItemResult(BaseModel):
    template_id: uuid.UUID
    success: bool
    esp_template_id: str | None = None
    error: str | None = None
    tokens_rewritten: int = 0

class BulkExportResponse(BaseModel):
    results: list[BulkExportItemResult]
    total: int
    succeeded: int
    failed: int
```

## Endpoints

### `POST /api/v1/connectors/sync/export`
- Auth: `require_role("developer")`
- Rate limit: `10/minute`
- Flow:
  1. Validate connection exists + belongs to project (BOLA)
  2. Fetch HTML from body or DB via `template_id`
  3. If `rewrite_tokens` and tokens detected and source ≠ target → `TokenRewriterService.rewrite()`
  4. `provider.create_template(name, html, credentials)`
  5. Return `ExportResponse`

### `POST /api/v1/connectors/sync/export-bulk`
- Auth: `require_role("developer")`
- Rate limit: `5/minute`
- Flow:
  1. Validate connection
  2. For each template_id (sequentially, with per-item error isolation):
     - Fetch HTML, rewrite tokens if needed, push to ESP
     - On error: capture in result, continue to next
  3. Return `BulkExportResponse` with per-item results
