### Step 1: Add `export_frame_screenshots()` to `FigmaDesignSyncService`

In `app/design_sync/figma/service.py`, add after `download_image_bytes()` (~line 1562):

```python
async def export_frame_screenshots(
    self,
    file_key: str,
    access_token: str,
    node_ids: list[str],
    *,
    scale: float = 2.0,
) -> dict[str, bytes]:
    """Export and download PNG screenshots for multiple frames in one call.

    Batches node_ids in groups of 100 (Figma API limit), then downloads
    all CDN images concurrently.

    Returns:
        Mapping of node_id → PNG bytes. Missing/failed nodes are omitted.
    """
    if not node_ids:
        return {}

    exported = await self.export_images(
        file_key, access_token, node_ids, format="png", scale=scale,
    )

    if not exported:
        return {}

    async def _download_safe(img: ExportedImage) -> tuple[str, bytes | None]:
        try:
            data = await self.download_image_bytes(img)
            return (img.node_id, data)
        except Exception:
            logger.warning(
                "design_sync.figma.frame_download_failed",
                node_id=img.node_id,
                exc_info=True,
            )
            return (img.node_id, None)

    results = await asyncio.gather(*[_download_safe(img) for img in exported])

    out: dict[str, bytes] = {}
    for node_id, data in results:
        if data is not None:
            out[node_id] = data

    logger.info(
        "design_sync.figma.frame_screenshots_exported",
        file_key=file_key,
        requested=len(node_ids),
        downloaded=len(out),
    )
    return out
```

**Imports needed:** None new — `asyncio`, `ExportedImage`, `logger` already in scope.

### Step 2: Refactor `_capture_design_image()` in `extract.py`

**⚠ KEEP THE EXISTING 4-ARG SIGNATURE** — `(file_key, token, node_id, output_dir)`. Do NOT add a `service` parameter. The function creates `FigmaDesignSyncService()` internally (line 127), and the callsite at `extract.py:308` passes 4 args. Tests patch the constructor.

Replace only the body (lines ~130-158), keeping constructor + dash→colon + `_get_scale()`:

```python
async def _capture_design_image(
    file_key: str,
    token: str,
    node_id: str,
    output_dir: Path,
) -> tuple[Path | None, int | None, int | None]:
    """Capture a single frame screenshot and save to output_dir/design.png."""
    from app.design_sync.figma.service import FigmaDesignSyncService

    figma_node_id = node_id.replace("-", ":")
    service = FigmaDesignSyncService()
    scale = _get_scale()

    try:
        screenshots = await service.export_frame_screenshots(
            file_key, token, [figma_node_id], scale=scale,
        )
        if figma_node_id not in screenshots:
            logger.warning("diagnose.image_capture_empty", node_id=node_id)
            return None, None, None

        image_bytes = screenshots[figma_node_id]
    except Exception:
        logger.warning("diagnose.image_capture_failed", node_id=node_id, exc_info=True)
        return None, None, None

    image_path = output_dir / "design.png"
    image_path.write_bytes(image_bytes)

    width, height = _read_png_dimensions(image_bytes)
    logger.info(
        "diagnose.image_captured",
        path=str(image_path),
        size_bytes=len(image_bytes),
        width=width,
        height=height,
    )
    return image_path, width, height
```
