MSO Conditional Patterns

This template demonstrates four key conditional comment patterns used for Outlook compatibility.

Version-Targeted Content

Left column content visible in all clients. Uses ghost table in Outlook 2007+ and inline-block div for modern clients.

Right column content. Properly nested MSO conditional closes the ghost table cell and opens the next one.

Outlook + IE Targeting

This block has extra padding/background in Outlook and IE. The combined conditional (gte mso 9)|(IE) targets both rendering engines.

This styled callout with a left border is only visible in non-Outlook clients. Outlook skips this entire block.
