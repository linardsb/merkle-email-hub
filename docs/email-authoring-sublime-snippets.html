Email Snippets — Consolidated Guide

Brand-agnostic snippets with tab-stop placeholders for any client

How to use

  1. Open an HTML file in Sublime Text
  2. Type the trigger word (e.g. email-base)
  3. Press Tab — the full HTML block expands
  4. Cursor lands on the first placeholder (font, color, etc.) — type your value
  5. Press Tab to jump to next placeholder, Shift+Tab to go back
  6. Press Esc when done editing placeholders

Tip: Go to Tools → Snippets in Sublime to browse all installed snippets.

Placeholder System

Every consolidated snippet uses tab-stop placeholders instead of hardcoded brand values:

${1:Helvetica, Arial, sans-serif} Font family
${2:#000000} Brand primary color
${3:#333333} Text color
${4:#ffffff} Background color
${5:images/logo.png} Logo / image URL
${6:Email Title} Title / heading text
${7:Button Text} CTA label
${8:#} Link URL

Press Tab to cycle through each placeholder. Type once — Sublime fills all matching stops.
