# Per-Operation Best Practices

## subject_line
**Goal:** Maximize open rate with honest, compelling preview

Techniques:
- Front-load the value proposition (first 30 chars matter most)
- Use power words: "new", "your", "introducing", "inside", "update"
- Create information gap without clickbait
- Test: "Would I open this?" — if no, rewrite

Quality checklist:
- [ ] Under 60 characters
- [ ] No ALL CAPS words
- [ ] No spam trigger phrases
- [ ] Accurately represents email content
- [ ] Works truncated at 30 chars (mobile)

## preheader
**Goal:** Complement the subject line, add second hook

Techniques:
- Continue the thought from the subject line
- Add specific detail the subject line couldn't fit
- Use as a "second subject line" for mobile
- Include social proof or urgency naturally

Quality checklist:
- [ ] 40-130 characters
- [ ] Does NOT repeat the subject line
- [ ] Adds new information or context
- [ ] Reads well as a continuation of the subject

## cta
**Goal:** Drive click with clear, benefit-focused action

Techniques:
- Start with a verb: Get, Start, Discover, Claim, Try, Join, Explore
- Focus on what the reader gets, not what they do
- Match CTA specificity to offer specificity
- Consider the reader's stage (awareness -> action)

Quality checklist:
- [ ] 2-5 words
- [ ] Starts with action verb
- [ ] Benefit-oriented (not "Click Here")
- [ ] Matches the email's primary offer

## body_copy
**Goal:** Persuade with scannable, engaging content

Techniques:
- Hook in first sentence (question, stat, bold claim)
- Short paragraphs (2-3 sentences max)
- One idea per paragraph
- "You" and "your" > "we" and "our"
- Social proof: numbers, testimonials, authority
- Single primary CTA per section

Quality checklist:
- [ ] Scannable (short paragraphs, clear hierarchy)
- [ ] Single primary CTA
- [ ] Reader-centric language
- [ ] Specific (numbers > vague claims)

## rewrite
**Goal:** Improve clarity and engagement without changing meaning

Techniques:
- Identify the core message first, then rebuild
- Replace passive voice with active
- Cut redundant words and phrases
- Strengthen verb choices (walked -> strode, said -> declared)
- Improve rhythm (vary sentence length)

Quality checklist:
- [ ] Core meaning preserved
- [ ] Clearer than original
- [ ] Similar length (+/-20%)
- [ ] Tone maintained

## shorten
**Goal:** Reduce word count 30-50% while keeping impact

Techniques:
- Remove qualifiers: very, really, quite, somewhat, actually
- Remove filler: "in order to" -> "to", "at this point in time" -> "now"
- Combine sentences that share a subject
- Cut the weakest sentence in each paragraph
- Replace phrases with single words

Quality checklist:
- [ ] 30-50% shorter
- [ ] Core message intact
- [ ] Key selling points preserved
- [ ] Still reads naturally

## expand
**Goal:** Add depth without becoming verbose

Techniques:
- Add a specific example for abstract claims
- Include a data point or statistic
- Add a brief testimonial or social proof
- Expand "what" into "what + why"
- Add a contrasting perspective ("Unlike X, this Y...")

Quality checklist:
- [ ] Adds genuine value (not padding)
- [ ] No unsupported claims introduced
- [ ] Tone maintained
- [ ] Each addition serves a purpose

## tone_adjust
**Goal:** Transform the register while preserving content

Techniques:
- Map vocabulary to target tone (see brand_voice.md axes)
- Adjust sentence length (casual = shorter, formal = longer)
- Change pronoun balance
- Modify punctuation style
- Replace idioms/metaphors appropriate to target tone

Common tones: professional, casual, urgent, friendly, authoritative, playful, empathetic

Quality checklist:
- [ ] All facts preserved
- [ ] Tone clearly matches target
- [ ] Vocabulary appropriate to tone
- [ ] Sentence structure matches tone
